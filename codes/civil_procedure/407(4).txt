Отказ в принятии к производству кассационной жалобы (протеста) Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСудья отказывает в принятии к производству кассационной жалобы (протеста), если: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) кассационная жалоба (протест) подана лицом, не имеющим права на обжалование (опротестование) судебного акта; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) кассационная жалоба (протест) подана на судебный акт, который в соответствии с законом не может быть обжалован (опротестован) в кассационном порядке; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) кассационная жалоба (протест) подана на судебный акт, который был рассмотрен в кассационном порядке; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа4) пропущен срок подачи кассационной жалобы (протеста), в восстановлении которого отказано; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа5) имеется определение о прекращении производства по кассационной жалобе (протесту) в связи с отказом (отзывом) лица, подавшим кассационную жалобу (протест). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ определении об отказе в принятии к производству кассационной жалобы (протеста) указываются основания для отказа в принятии к производству кассационной жалобы (протеста), решается вопрос о возврате государственной пошлины, уплаченной при подаче кассационной жалобы. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНа определение об отказе в принятии к производству кассационной жалобы (протеста) может быть подана ревизионная жалоба (протест). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае отмены определения об отказе в принятии к производству кассационной жалобы (протеста), жалоба (протест) считается поданной в день первоначального обращения в суд.
