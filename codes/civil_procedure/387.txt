Документы, прилагаемые к апелляционной жалобе (протесту) Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК апелляционной жалобе прилагаются: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадокумент, подтверждающий уплату государственной пошлины и почтовых расходов; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав случае подписания апелляционной жалобы представителем — документ, подтверждающий полномочия на ее подписание. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаАпелляционная жалоба (протест) предоставляется в суд с копиями по числу лиц, участвующих в деле, за исключением апелляционной жалобы (протеста), направляемой в виде электронного документа. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ необходимых случаях судья может обязать лицо, подающее апелляционную жалобу (протест), предоставить копии приложенных к апелляционной жалобе (протесту) письменных материалов по числу лиц, участвующих в деле. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК апелляционному протесту по делу, рассмотренному без участия прокурора, прилагается копия обращения лиц, указанных в части первой статьи 383 настоящего Кодекса, послужившего основанием к принесению данного протеста. За исключением дел, по которым законом предусмотрено участие прокурора, но рассмотренным без его надлежащего извещения о времени и месте судебного разбирательства. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК апелляционной жалобе (протесту) на определение суда о возвращении заявления либо об отказе в принятии заявления должны быть также приложены возвращенное заявление и документы, приложенные к нему при подаче в суд. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаАпелляционная жалоба (протест) на решение суда о восстановлении на работе работника, с которым трудовой договор был незаконно прекращен, или работника, незаконно переведенного на другую работу, принимается лишь при условии приложения к ней документа об исполнении данного решения.
