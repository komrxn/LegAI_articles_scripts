Основания оставления заявления без рассмотрения Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСуд оставляет заявление без рассмотрения, если: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) заявление подано недееспособным лицом; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) заявление от имени заинтересованного лица подано лицом, не имеющим полномочий на ведение дела; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) в производстве суда по гражданским делам, экономического суда либо третейского суда имеется дело по спору между теми же сторонами, о том же предмете и по тем же основаниям; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа4) между сторонами заключено третейское соглашение о передаче данного спора на разрешение третейского суда; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа5) стороны, не просившие о разбирательстве дела в их отсутствие, не явились без уважительных причин по вторичному вызову, а суд не считает возможным разрешить дело по имеющимся в деле материалам; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа6) истец, не просивший о разбирательстве дела в его отсутствие, не явился в суд по вторичному вызову, а ответчик не требует разбирательства дела по существу; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа7) во время разбирательства дела особого производства возник спор о праве, подведомственный суду; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа8) муж предъявил иск о расторжении брака без согласия жены во время ее беременности или в течение года после рождения ребенка; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа9) истцом подано заявление об оставлении заявления без рассмотрения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа91) истец, в интересах которого прокурором, государственным органом, организацией и иным лицом предъявлен иск, отказался от иска и при этом не затрагиваются права, свободы и законные интересы третьих лиц; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа10) истцом не соблюден досудебный порядок урегулирования спора с ответчиком, когда это предусмотрено законом или договором; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа101) истцом не соблюден порядок урегулирования спора с проведением процедуры медиации с ответчиком, когда это предусмотрено законом для данной категории споров или договором; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа102) стороны, заявившие ходатайство о проведении процедуры медиации, по истечении срока ее проведения не явились на заседание суда без уважительных причин; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа103) между сторонами заключено медиативное соглашение; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа11) заявление о признании имущества (вещи) бесхозяйным подано ранее срока, установленного законом, или с нарушением предусмотренного законом порядка выявления и учета имущества (вещи) бесхозяйным.
