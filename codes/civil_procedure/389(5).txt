Прекращение производства по апелляционной жалобе (протесту) Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСуд апелляционной инстанции прекращает производство по апелляционной жалобе (протесту), если: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) апелляционная жалоба (протест) подана на судебный акт, который в соответствии с настоящим Кодексом не может быть обжалован (опротестован), и была ошибочно принята к производству; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) апелляционная жалоба (протест) подана на судебный акт, который был рассмотрен в апелляционном порядке, и ошибочно принята к производству; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) апелляционная жалоба (протест) подана лицом, не имеющим права обжаловать (опротестовать) судебный акт; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа4) после принятия апелляционной жалобы к производству от лица, подавшего апелляционную жалобу, поступило заявление об отказе от апелляционной жалобы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа5) после принятия апелляционного протеста к производству от прокурора, принесшего протест, или вышестоящего прокурора поступило заявление об отзыве протеста; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа6) до принятия судебного акта по апелляционной жалобе (протесту) юридическое лицо, являющееся стороной в деле, ликвидировано; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа7) до принятия судебного акта по апелляционной жалобе (протесту) гражданин, являющийся стороной в деле, умер, а спорное правоотношение не допускает правопреемства. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаО прекращении производства по апелляционной жалобе (протесту) выносится определение, в котором разрешается вопрос распределения между сторонами судебных расходов. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае прекращения производства по апелляционной жалобе (протесту) повторное обращение того же лица по тем же основаниям в суд с апелляционной жалобой (протестом) не допускается.
