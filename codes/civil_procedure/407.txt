Документы, прилагаемые к кассационной жалобе (протесту) Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК кассационной жалобе прилагаются: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадокумент, подтверждающий уплату государственной пошлины и почтовых расходов; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав случае подписания кассационной жалобы представителем — документ, подтверждающий полномочия на ее подписание. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаКассационная жалоба (протест) предоставляется в суд с копиями по числу лиц, участвующих в деле, за исключением кассационной жалобы (протеста), направляемой в виде электронного документа. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ необходимых случаях судья может обязать лицо, подающее кассационную жалобу (протест), предоставить копии приложенных к кассационной жалобе (протесту) письменных материалов по числу лиц, участвующих в деле. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК кассационному протесту по делу, рассмотренному без участия прокурора, прилагается копия обращения лиц, указанных в части первой статьи 403 настоящего Кодекса, послужившего основанием к принесению данного протеста, за исключением дел, по которым законом предусмотрено участие прокурора, но рассмотренным без его надлежащего извещения о времени и месте судебного разбирательства. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК кассационной жалобе (протесту) на определение суда о возвращении заявления либо об отказе в принятии заявления должны быть также приложены возвращенное заявление и документы, приложенные к нему при подаче в суд. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаКассационная жалоба (протест) на решение суда о восстановлении на работе работника, с которым трудовой договор был незаконно прекращен, или работника, незаконно переведенного на другую работу, принимается лишь при условии приложения к ней документа об исполнении данного решения.
