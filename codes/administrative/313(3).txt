Статья 313(3) КоАО РУз
Порядок направления дела об административных правонарушениях в суд Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаКомпетентный орган (должностное лицо), получив заявление о примирении потерпевшего или его законного представителя, в трехдневный срок, с согласия правонарушителя, выносит постановление о направлении дела в суд в связи с примирением сторон и направляет дело по подсудности. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ описательной части постановления указывается: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаоснование для производства дела об административном правонарушении; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документасведения о лицах, совершивших административное правонарушение, и совершенных ими административных правонарушениях; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документасодержание заявления о примирении и отношение правонарушителя к заявлению. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ резолютивной части постановления указывается: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документарешение о направлении дела в суд; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаинформация о вещественных, письменных и цифровых доказательствах. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли по делу привлечено в качестве правонарушителей несколько лиц и не со всеми достигнуто примирение, то материалы в отношении правонарушителей, с которыми не достигнуто примирение, выделяются и производство по ним осуществляется с соблюдением общих правил. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОбстоятельства, предусмотренные частью четвертой настоящей статьи, должны быть указаны в постановлении компетентного органа (должностного лица) о направлении дела в суд в связи с примирением сторон.
