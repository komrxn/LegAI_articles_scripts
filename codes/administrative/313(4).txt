Статья 313(4) КоАО РУз
Судебное разбирательство Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСудебное разбирательство по делу о примирении должно быть рассмотрено не позднее трех дней со дня его поступления в суд. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ судебном заседании принимают участие: правонарушитель, потерпевший, их законные представители и адвокат, если последний участвует в деле. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСудебное разбирательство по делу о примирении начинается с оглашения заявления потерпевшего о примирении. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСуд заслушивает правонарушителя и потерпевшего об обстоятельствах совершенного административного правонарушения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСуд определяет: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадобровольность примирения и его мотивы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадобровольность признания вины правонарушителем; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаосознал ли правонарушитель последствие совершенного им деяния и принял ли меры по заглаживанию причиненного ущерба; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документане оказывалось ли на потерпевшего или правонарушителя какое-либо давление; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документавопросы, связанные с возмещением причиненного ущерба; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документасогласие правонарушителя, потерпевшего и их законных представителей на примирение. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗатем суд заслушивает мнение адвоката, если он участвует в судебном заседании. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли в ходе судебного заседания будут установлены недобровольность примирения, признания вины, отказ от возмещения ущерба, то суд отменяет постановление о направлении дела в суд в связи с примирением сторон и рассматривает дело на общих основаниях. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОднако при наличии в совершенном деянии признаков другого административного правонарушения либо состава преступления суд отменяет постановление о направлении дела в суд в связи с примирением сторон и направляет дело органу, оформившему протокол об административном правонарушении или возбудившему дело об административном правонарушении, для рассмотрения в общем порядке или для проведения доследственной проверки, дознания и предварительного расследования. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПо результатам судебного разбирательства суд выносит постановление в порядке, установленном законом. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВо время судебного заседания ведется протокол по правилам, предусмотренным статьей 308 настоящего Кодекса.
