Форма и содержание заявления о пересмотре судебного акта по вновь открывшимся обстоятельствам Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗаявление о пересмотре вступившего в законную силу судебного акта по вновь открывшимся обстоятельствам подается в суд в письменной форме. Заявление подписывается лицом, подающим его, или его представителем. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ заявлении о пересмотре вступившего в законную силу судебного акта по вновь открывшимся обстоятельствам должны быть указаны: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) наименование суда, в который подается заявление; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) лица (фамилия, имя, отчество), участвующие в деле, их местонахождение (почтовый адрес) или местожительство; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) наименование суда, вынесшего судебный акт, пересмотр которого по вновь открывшимся обстоятельствам требует заявитель; номер дела, дата вынесения судебного акта; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа4) вновь открывшееся обстоятельство, являющееся по мнению заявителя основанием для пересмотра вступившего в законную силу судебного акта, со ссылкой на документы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа5) требование лица, подающего заявление; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа6) перечень прилагаемых документов. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК заявлению должны быть приложены: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) копии документов, подтверждающих вновь открывшиеся обстоятельства; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) копия судебного акта, о пересмотре которого требует заявитель; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) в случае подписания заявления представителем — документ, подтверждающий полномочия на ее подписание.
