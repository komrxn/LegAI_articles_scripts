Условия, при которых обмен жилыми помещениями не допускается Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОбмен жилого помещения не допускается, если: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документак нанимателю предъявлен иск о расторжении или изменении договора найма жилого помещения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документана жилое помещение наложен арест (запрещение отчуждения) в связи с возбуждением уголовного дела; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаобмен носит фиктивный характер; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документажилое помещение находится в технически неисправном состоянии, подлежит сносу или переоборудованию для использования в других целях либо передается для государственных или общественных нужд; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документажилое помещение подлежит капитальному ремонту с переустройством или перепланировкой жилых помещений; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаодно из обмениваемых жилых помещений находится в доме коммунального жилищного фонда целевого назначения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапомещение является служебным или находится в общежитии; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав связи с обменом жилищные условия одной из обменивающихся сторон существенно ухудшаются, в результате чего граждане становятся нуждающимися в улучшении жилищных условий.
