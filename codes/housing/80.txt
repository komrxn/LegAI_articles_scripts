Выселение из служебных жилых помещений с предоставлением другого жилого помещения Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВыселяются из служебных жилых помещений с предоставлением другого жилого помещения: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаинвалиды I и II групп; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаветераны войны 1941—1945 годов и воины-интернационалисты; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документасемьи военнослужащих, погибших или пропавших без вести при исполнении ими военной службы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документасемьи, имеющие детей-инвалидов; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документародители-воспитатели в случае ликвидации семейного детского дома; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документалица, проработавшие (прослужившие) в организациях, предоставивших им служебное жилое помещение, не менее десяти лет; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документалица, освобожденные от должности, в связи с которой им было предоставлено жилое помещение, но не прекратившие трудовых отношений с предприятием, учреждением, организацией, предоставившими это помещение; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документалица, с которыми прекращен трудовой договор в связи с изменением численности (штата) работников или изменением характера работы либо ликвидацией предприятия, учреждения, организации; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаработники, вышедшие на пенсию с предприятий, учреждений, организаций, на которых они работали; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документачлены семьи умершего лица, которому было предоставлено служебное жилое помещение; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаодинокие лица с проживающими вместе с ними несовершеннолетними детьми.
