Статья 48(1) УК РУз
Ограничение свободы Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОграничение свободы состоит в установлении судом в отношении осужденного полного запрета на покидание жилища под тем или иным предлогом, либо ограничения на выход из жилища кроме времени работы или учебы, а также времени необходимого для поездки до места работы или учебы и для возвращения обратно в жилище. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОграничение свободы назначается на срок от одного месяца до пяти лет и отбывается под надзором органов, определяемых судом. Ограничение свободы, условия его отбывания по месту жительства осужденного определяются судом с учетом характера совершенного деяния и предотвращения уклонения от исполнения вынесенного решения суда. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаС учетом особенностей применяемого запрета (ограничения), суд может возложить на осужденного следующие дополнительные запреты (ограничения): Предложения по документуПрослушать аудиоПолучить ссылку из элемента документане посещать определенные места; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документане участвовать в проведении массовых и иных мероприятий; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документане заниматься определенной деятельностью; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документане владеть или не хранить у себя определенные предметы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документане управлять транспортным средством; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документане изменять место жительства, место работы и (или) учебы, не выезжать за пределы соответствующей административной территории без согласия органа, осуществляющего надзор за осужденными; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документане устанавливать контакты с определенными лицами; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документане использовать средства связи, в том числе Интернет; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документане употреблять алкогольные напитки. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСуд может возложить на осужденного к ограничению свободы обязанности возместить причиненный им материальный и моральный ущерб, устроиться на работу или учебу, а также иные обязанности, способствующие его исправлению. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае, если осужденный к ограничению свободы в период отбывания наказания осознал свои преступные деяния, твердо встал на путь исправления, возместил причиненный материальный и моральный ущерб, суд может отменить полностью или частично ранее установленные осужденному запреты (ограничения). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае злостного уклонения осужденного от отбывания наказания в виде ограничения свободы, а также невыполнения им возложенных на него судом обязанностей, суд может заменить неотбытый срок ограничения свободы другим видом наказания. Время уклонения в срок отбытого наказания не засчитывается. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОграничение свободы не назначается военнослужащим, иностранным гражданам, а также лицам, не имеющим постоянного места жительства в Республике Узбекистан.
