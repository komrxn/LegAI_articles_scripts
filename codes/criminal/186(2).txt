Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаТе же действия, совершенные: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаа) в крупном размере; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаб) по предварительному сговору группой лиц; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав) повторно или опасным рецидивистом; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаг) путем злоупотребления должностным положением, — Предложения по документуПрослушать аудиоПолучить ссылку из элемента документанаказываются штрафом от трехсот до шестисот базовых расчетных величин или лишением свободы от трех до семи лет. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаТе же действия, совершенные: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаа) в особо крупном размере; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаб) организованной группой или в ее интересах, — Предложения по документуПрослушать аудиоПолучить ссылку из элемента документанаказываются лишением свободы от семи до десяти лет.
