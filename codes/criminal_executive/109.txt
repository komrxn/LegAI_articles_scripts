Особенности применения взыскания в виде водворения осужденных в дисциплинарное отделение Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК дисциплинарным отделениям учреждений по исполнению наказания относятся: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаизоляторы колоний-поселений и воспитательных колоний; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаизоляторы и карцеры колоний общего, строгого и особого режимов; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документакарцеры следственных изоляторов и тюрем. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОсужденные водворяются в изоляторы и карцеры на срок до пятнадцати суток, а в изоляторы воспитательных колоний — до десяти суток. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ карцеры колоний общего, строгого и особого режимов переводятся осужденные — злостные нарушители порядка отбывания наказания на срок до шести месяцев, а в колониях для содержания женщин — до трех месяцев. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОсужденным, водворенным в изоляторы, запрещаются свидания, телефонные разговоры, приобретение продуктов питания и предметов первой необходимости, получение и отправление посылок, передач и бандеролей, отправление писем и другой корреспонденции. Им не разрешается курить, пользоваться настольными играми. Осужденные имеют право пользоваться ежедневной прогулкой продолжительностью полтора часа. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОсужденным в карцерах запрещаются свидания, телефонные разговоры, получение и отправление посылок, передач и бандеролей, пользование настольными играми. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОсужденные, переведенные в карцеры колоний общего, строгого и особого режимов, имеют право: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документарасходовать на приобретение продуктов питания и предметов первой необходимости деньги до 26,5 процентов от установленного минимального размера оплаты труда; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапользоваться ежедневной прогулкой продолжительностью два часа. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ карцерах колоний особого режима, следственных изоляторов и тюрем устанавливается одиночное содержание осужденных. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВодворение и перевод осужденных в дисциплинарное отделение производятся по постановлению начальника учреждения с указанием срока и при наличии медицинского разрешения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае перевода осужденных из дисциплинарных отделений в лечебно-профилактические учреждения, срок их нахождения в лечебных учреждениях засчитывается в срок отбывания взыскания. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНе водворяются в дисциплинарные отделения: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документалица с инвалидностью I группы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаженщины, освобожденные от работы по беременности и родам, а также имеющие при себе грудных детей. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОсужденные, содержащиеся в дисциплинарных отделениях, к труду не привлекаются.
