Меры поощрения Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК  осужденным к лишению свободы меры поощрения применяются на основании критериев, предусмотренных частью первой статьи 971 настоящего Кодекса. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК осужденным к лишению свободы применяются следующие меры поощрения: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаобъявление благодарности; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадосрочное снятие ранее примененного дисциплинарного взыскания; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документанаграждение грамотой, ценным подарком или деньгами; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаразрешение на получение дополнительной посылки, передачи или бандероли; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапредоставление права на дополнительное свидание или телефонный разговор; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаувеличение суммы денег на приобретение продуктов питания и предметов первой необходимости; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаперевод на улучшенные условия содержания, за исключением лиц, содержащихся в колониях-поселениях. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПоощрение в виде перевода на улучшенные условия содержания дает право осужденным на дополнительное расходование денег в соответствии с частью третьей статьи 75 настоящего Кодекса, а также право: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав колониях общего, строгого режимов и в воспитательных колониях — дополнительно получать в течение года одно свидание, один телефонный разговор, одну посылку или передачу; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав колониях особого режима — перевода из камер в общие жилые помещения, с дополнительным получением в течение года одного краткосрочного свидания, одного телефонного разговора, одной бандероли; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав тюрьмах — на дополнительное получение в течение года одного краткосрочного свидания, одной посылки или передачи.
