Дополнительные права для осужденных беременных женщин и женщин, имеющих детей Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОсужденные беременные женщины и женщины, имеющие детей в возрасте до трех лет, кроме прав, предусмотренных настоящим Кодексом для осужденных, имеют также право: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапроживать за пределами территории учреждения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапомещать детей в дома ребенка при колониях по исполнению наказания; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документавыезжать за пределы учреждения для устройства детей и свидания с ними. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОсужденным женщинам предоставляются в течение года четыре длительных свидания с несовершеннолетними детьми продолжительностью до пяти суток, а содержащимся в колониях-поселениях длительные свидания предоставляются без ограничения, в свободное от работы время, с возможностью проживания за пределами территории учреждения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНа свидание к осужденным женщинам допускаются несовершеннолетние дети в сопровождении одного близкого родственника либо опекуна (попечителя) при предъявлении документов, удостоверяющих их личность, подтверждающих их родство либо опекунство (попечительство). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСвидания осужденных женщин с несовершеннолетними детьми не засчитываются в количество свиданий, установленных настоящим Кодексом. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСвидания осужденных женщин с несовершеннолетними детьми, находящимися в детских домах за пределами учреждения по исполнению наказания, производятся по просьбе осужденных женщин
