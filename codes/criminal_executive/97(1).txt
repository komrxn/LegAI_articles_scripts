Критерии определения становления осужденных на путь исправления Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСтановление осужденных, отбывающих наказание в учреждениях по исполнению наказания, на путь исправления определяется комиссией, предусмотренной статьей 1021 настоящего Кодекса, по итогам каждого квартала на основе одного из следующих критериев: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадобросовестное отношение осужденного к труду, выполнение установленных норм труда, осознанное формирование навыков самостоятельной работы, заинтересованность в обучении и приобретении профессии; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаучастие осужденного в культурно-просветительских мероприятиях, организуемых в учреждении по исполнению наказания, его инициативность, стремление быть примером для других осужденных; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документасоблюдение осужденным установленных правил внутреннего распорядка учреждения по исполнению наказания, применение мер поощрения к осужденному, неприменение в отношении него дисциплинарного взыскания. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗаключение о признании осужденного вставшим или не вставшим на путь исправления оформляется комиссией и приобщается к личному делу осужденного. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаС осужденными, признанными комиссией не вставшими на путь исправления, проводится воспитательно-профилактическая работа на основе отдельного плана с привлечением психолога учреждения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПризнанный не вставшим на путь исправления осужденный вправе обжаловать заключение комиссии в порядке, установленном статьей 107 настоящего Кодекса.
