Регистрация расторжения брака при взаимном согласии супругов Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри расторжении брака по взаимному согласию супругов заявители в своем заявлении органу записи актов гражданского состояния обязаны подтвердить отсутствие у них общих несовершеннолетних детей и имущественных споров. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРегистрация расторжения брака производится по истечении трех месяцев со дня подачи заявления в орган записи актов гражданского состояния независимо от места регистрации по месту постоянного проживания или временного пребывания заявителей. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОрганы записи актов гражданского состояния не позднее трех дней со дня подачи заявления должны письменно уведомить примирительную комиссию схода граждан по месту совместного жительства супругов для принятия соответствующих мер по их примирению, если они не проживают совместно — примирительную комиссию схода граждан по месту жительства каждого из них.
