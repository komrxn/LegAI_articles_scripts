Вопросы, разрешаемые судом при вынесении решения о расторжении брака Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри расторжении брака в судебном порядке супруги могут представить на рассмотрение суда соглашение о том, с кем из них будут проживать несовершеннолетние дети, о порядке выплаты средств на содержание детей и (или) нетрудоспособного нуждающегося супруга, о размерах этих средств либо о разделе общего имущества супругов. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри отсутствии соглашения между супругами по вопросам, указанным в части первой настоящей статьи, а также при установлении, что данное соглашение нарушает интересы детей или одного из супругов, суд обязан: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаопределить, с кем из родителей будут проживать несовершеннолетние дети после расторжения брака; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаопределить, с кого из родителей и в каких размерах будут взыскиваться алименты на содержание несовершеннолетних детей; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапроизвести по требованию супругов (одного из них) раздел имущества, находящегося в их совместной собственности; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаопределить по требованию супруга, имеющего право на получение содержания от другого супруга, размер этого содержания. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае, если раздел имущества затрагивает интересы третьих лиц, суд должен выделить требование о разделе имущества в отдельное производство. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри рассмотрении дела о расторжении брака требования о взыскании средств, израсходованных на проведение свадьбы, не подлежат удовлетворению.
