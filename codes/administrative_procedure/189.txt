Решение суда по делу об обжаловании решения, действий (бездействия) административного органа, органа самоуправления граждан, их должностных лиц Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРешение по делу об обжаловании решений, действий (бездействия) административных органов, органов самоуправления граждан, их должностных лиц принимается судом по правилам, установленным в главе 19 настоящего Кодекса. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСуд, установив, что обжалуемое решение или отдельные его части либо действия (бездействие) противоречат законодательству и нарушает права и охраняемые законом интересы заявителя, принимает решение о признании решения или его отдельных частей недействительными либо действий (бездействия) незаконными. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае, если суд установит, что обжалуемое решение или отдельные его положения либо действия (бездействие) соответствуют законодательству и не нарушает права и охраняемые законом интересы заявителя, принимает решение об отказе в удовлетворении заявленного требования. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ резолютивной части решения об удовлетворении заявления о признании недействительным решения, незаконными действий (бездействия) должно быть указано: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) наименование органа (должностного лица), принявшего обжалуемое решение либо совершившего действия (бездействие), название, номер, дата принятия решения, дата и место совершения обжалуемых действий (бездействия); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) признание обжалуемого решения полностью или в части недействительным или действий (бездействия) незаконными; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) распределение судебных расходов. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае признания решения недействительным, действий (бездействия) незаконными, суд обязывает соответствующий орган или должностное лицо: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапринять решение или совершить определенные действия в соответствии с законом либо устранить иным способом допущенные нарушения прав, свобод и законных интересов заявителя; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаустранить допущенные нарушения и сообщить об исполнении решения в суд и заявителю в течение одного месяца со дня вступления решения суда в законную силу, если иной срок не установлен судом. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРешение суда, принятое по результатам рассмотрения дела, вступает в законную силу по истечении одного месяца с момента его принятия, если оно не обжаловано (опротестовано) в апелляционном порядке.
