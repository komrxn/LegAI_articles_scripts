Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНарушением или неправильным применением норм материального права являются: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) неприменение закона или иного акта законодательства, подлежащего применению; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) применение закона или иного акта законодательства, не подлежащего применению; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) неправильное истолкование закона или иного акта законодательства. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНарушение или неправильное применение норм процессуального права является основанием для изменения или отмены решения суда первой инстанции, если это привело либо могло привести к принятию неправильного решения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРешение суда может быть отменено, если: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) нарушено правило о языке, на котором ведется судопроизводство, при рассмотрении дела; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) протокол судебного заседания не подписан или подписан не теми лицами, которые указаны в части второй статьи 176 настоящего Кодекса, либо к протоколу судебного заседания не приобщены электронные или иные носители аудио- и видеозаписи, если проведена аудио- или видеозапись судебного заседания; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) нарушено правило о тайне совещания судей при принятии решения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОснованиями для отмены решения суда первой инстанции в связи с нарушением или неправильным применением норм процессуального права в любом случае являются: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) рассмотрение дела судом в незаконном составе; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) рассмотрение дела в отсутствие кого-либо из участвующих в деле лиц, надлежаще не извещенных о времени и месте судебного заседания; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) принятие судом решения о правах и обязанностях лиц, не привлеченных к участию в деле; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа4) не подписание решения судьей или одним из судей, если дело рассмотрено в коллегиальном составе судей, либо подписание решения не теми судьями, которые указаны в решении; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа5) отсутствие в деле протокола судебного заседания; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа6) непринятие судом решения по заявленному требованию; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа7) рассмотрение дела с нарушением правил подведомственности.
