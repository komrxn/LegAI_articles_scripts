Содержание решения Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРешение суда состоит из вводной, описательной, мотивировочной и резолютивной частей. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВо вводной части решения указывается наименование суда, принявшего решение; состав суда, секретарь судебного заседания; номер дела, дата и место принятия решения; предмет требования; лица, участвующие в деле, а также их представители и иные участники судебного процесса. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОписательная часть решения должна содержать краткое изложение заявленных требований и возражений, объяснений, заявлений и ходатайств лиц, участвующих в деле. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ мотивировочной части решения должны быть указаны: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документафактические обстоятельства дела, установленные судом; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадоказательства, на которых основаны выводы суда об обстоятельствах дела; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаоснования, по которым суд отверг те или иные доказательства, принял или отклонил доводы лиц, участвующих в деле; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаакты законодательства, которыми руководствовался суд при принятии решения, и основания, по которым суд не применил акты законодательства, на которые ссылались лица, участвующие в деле. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ мотивировочной части решения могут содержаться ссылки на постановления Пленума Верховного суда Республики Узбекистан. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРезолютивная часть решения должна содержать выводы об удовлетворении или отказе в удовлетворении полностью или в части каждого из заявленных требований. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае удовлетворения требования о возмещении убытков в резолютивной части решения указывается сумма, подлежащая возмещению. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли суд установил порядок исполнения решения или принял меры по обеспечению его исполнения, об этом указывается в резолютивной части решения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ резолютивной части решения указывается о распределении между лицами, участвующими в деле, судебных расходов, срок и порядок обжалования (опротестования) решения.
