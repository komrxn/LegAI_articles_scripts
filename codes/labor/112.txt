Случаи, когда с работником заключается срочный трудовой договор Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСрочный трудовой договор заключается: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документана время исполнения обязанностей отсутствующего работника, за которым в соответствии с трудовым законодательством и иными правовыми актами о труде, трудовым договором сохраняется место работы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документана время выполнения временных (до двух месяцев) работ; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадля выполнения сезонных работ, когда в силу природных условий работа может производиться только в течение определенного периода (сезона); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадля проведения работ, выходящих за рамки обычной деятельности работодателя (реконструкция, монтажные, пусконаладочные и другие работы), а также работ, связанных с заведомо временным (до одного года) расширением производства или увеличением объема работ (продукции, услуг); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документас лицами, поступающими на работу в организации, созданные на заведомо определенный период согласно учредительным документам; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документас лицами, принимаемыми для выполнения заведомо определенной работы, носящей срочный характер, в случаях, когда ее завершение не может быть определено конкретной датой; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадля выполнения работ, непосредственно связанных с договором производственного обучения, прохождения оплачиваемой производственной практики или стажировки; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документас лицами, направленными органами по труду на работы временного характера или оплачиваемые общественные работы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документас гражданами, направленными для прохождения альтернативной службы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документас иностранными гражданами и лицами без гражданства, законно въехавшими в Республику Узбекистан для осуществления трудовой деятельности на ее территории. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСрочный трудовой договор заключается и в других случаях в соответствии с настоящим Кодексом или иным законом.
