Предложения по документуПрослушать аудиоПолучить ссылку из элемента документанорма численности работников — численность работников определенного профессионально-квалификационного состава, требующаяся для выполнения производственного задания в определенных организационно-технических условиях; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документанорма обслуживания — установленное количество единиц оборудования, площадей территорий и объектов оказания услуг, которое должно обслуживаться одним работником или группой работников соответствующей численности и квалификации при определенных организационно-технических условиях в течение смены; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документанорма введения отдельных должностей — устанавливаемая для бюджетных организаций соответствующим органом государственного управления численность работников, необходимых для выполнения отдельных задач и функций организации; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документанорма управляемости — количество работников, которое должно быть непосредственно подчинено одному руководителю; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаквалификационная норма — необходимый уровень квалификации (знаний, умений, навыков) для выполнения работником трудовой функции; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадругие нормы, определяющие необходимые затраты труда. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ условиях коллективных форм организации и оплаты труда могут применяться укрупненные нормы труда, устанавливаемые на конкретный объем продукции, технологический процесс в целом или определенный объем работ (услуг), а также комплексные нормы труда, устанавливаемые по конечному укрупненному измерителю (кубический метр, квадратный метр, тысяча единиц продукции и другие), характеризующему комплекс взаимосвязанных работ, исчисленные на базе принятых норм труда.
