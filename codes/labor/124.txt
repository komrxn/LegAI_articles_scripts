Документы, требуемые при приеме на работу Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри приеме на работу лицо, поступающее на работу, предъявляет следующие документы: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапаспорт либо заменяющий его документ или идентификационную ID-карту, а лица в возрасте до шестнадцати лет — свидетельство о рождении или идентификационную ID-карту; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатрудовую книжку в бумажной форме или выписку из электронной трудовой книжки, заверенную по последнему месту работу, за исключением лиц, поступающих на работу впервые. Лица, поступающие на работу по совместительству, вместо трудовой книжки предъявляют справку установленного образца с основного места работы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документавоенный билет либо приписное удостоверение соответственно для военнообязанных или призывников; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадиплом об окончании высшей или средней специальной, профессиональной образовательной организации, удостоверение (сертификат) на право выполнения данной работы либо иной соответствующий документ при поступлении на работу, к выполнению которой могут быть допущены только лица, имеющие специальное образование или специальную подготовку; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаперсональный идентификационный номер физического лица. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаГосударственные органы и организации при приеме на работу самостоятельно запрашивают сведения о документах, предусмотренных абзацами вторым и третьим части первой настоящей статьи, у соответствующих государственных органов и организаций, в том числе через межведомственные интеграционные платформы в информационных системах. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРаботодатель обязан: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаоформить трудовую книжку на проработавших более пяти дней лиц, поступивших на работу впервые, и на лиц ранее работавших, в отношении которых законодательство не предусматривает ведение трудовых книжек; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапринять меры для постановки на учет в накопительной пенсионной системе работников, ранее не состоявших на таком учете. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри приеме на работу запрещается требовать от лица, поступающего на работу, документы, не предусмотренные настоящим Кодексом, а также другими актами законодательства.
