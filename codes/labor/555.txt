Решение комиссии по трудовым спорам Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаКомиссия по трудовым спорам принимает решение по соглашению представителей профсоюзного комитета и работодателя. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРешение комиссии по трудовым спорам должно быть мотивированным и основанным на трудовом законодательстве и иных правовых актах о труде, трудовом договоре. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ решении комиссии по трудовым спорам указываются: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документанаименование организации, а в случае, когда индивидуальный трудовой спор рассматривается комиссией по трудовым спорам обособленного подразделения организации, — наименование подразделения, фамилия, имя, отчество, должность, профессия или специальность обратившегося в комиссию работника; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадаты обращения в комиссию и рассмотрения спора, суть спора; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документафамилии, имена, отчества членов комиссии и других лиц, присутствовавших на заседании; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документасуть решения и его обоснование (со ссылкой на законодательство и иные правовые акты о труде, трудовой договор); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документарезультаты голосования. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ решении комиссии по трудовым спорам по денежным требованиям должна быть указана точная сумма, причитающаяся работнику. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРешение комиссии по трудовым спорам подписывается председателем и секретарем заседания комиссии, имеет обязательную силу и какому-либо утверждению не подлежит. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаКопия решения комиссии по трудовому спору вручается (направляется) работнику, работодателю и профсоюзному комитету в трехдневный срок со дня принятия решения.
