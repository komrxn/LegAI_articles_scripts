Статья 280 ТК РУз
Обязанность работодателя производить гарантийные выплаты работнику Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРаботодатель обязан производить гарантийные выплаты за период: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаучастия работника в коллективных переговорах, подготовке проекта коллективного договора; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаисполнения обязанностей по представительству и защите интересов работников лицами, являющимися представителями работников и не освобожденными от производственной работы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаучастия в работе комиссии по трудовым спорам члена этой комиссии; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаотстранения работника от работы, в случае если он был отстранен в связи с непрохождением обучения и проверки знаний и навыков в области охраны труда или обязательного медицинского осмотра не по своей вине; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаотстранения работника от работы в связи с проведением в отношении него служебного расследования; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документанахождения работника в ежегодном трудовом отпуске; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапростоя и брака, допущенного по вине работодателя; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаневыполнения норм труда, неисполнения трудовых (должностных) обязанностей по вине работодателя; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаперевода на более легкую или исключающую воздействие неблагоприятных производственных факторов работу по состоянию здоровья в соответствии со статьей 364 настоящего Кодекса; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документавыполнения уполномоченным лицом по охране труда возложенных на него функций; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документанахождения работника в служебной командировке; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапереезда работника с его согласия вместе с работодателем или по предложению работодателя в другую местность на время, необходимое для переезда и обустройства работника на новом месте, но не более шести рабочих дней; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапрохождения работником с отрывом от работы обязательной для него и работодателя переподготовки или обязательного повышения квалификации; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документанахождения работника в учебных отпусках; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документанахождения работника в творческих отпусках; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатрудоустройства при прекращении трудового договора по отдельным основаниям; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапрохождения работниками обязательных медицинских осмотров; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаперевода беременной женщины на более легкую либо исключающую воздействие неблагоприятных производственных факторов работу; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаперевода одного из родителей (опекуна), осуществляющего уход за ребенком в возрасте до двух лет, на более легкую либо исключающую воздействие неблагоприятных производственных факторов работу; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документавынужденного прогула работника, вызванного незаконным отказом в приеме на работу, незаконным переводом работника на другую работу, незаконным отстранением работника от работы, незаконным прекращением трудового договора, а также неисполнением решения органа по рассмотрению индивидуальных трудовых споров о восстановлении работника на прежней работе. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПомимо гарантийных выплат, предусмотренных частью первой настоящей статьи, работодатель обязан производить работникам другие гарантийные выплаты, установленные настоящим Кодексом, законодательством, коллективными соглашениями, коллективным договором, а также локальными актами, принимаемыми работодателем по согласованию с профсоюзным комитетом, трудовым договором.
