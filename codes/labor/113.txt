Случаи, когда с работником может заключаться срочный трудовой договор Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПо соглашению между работником и работодателем срочный трудовой договор может заключаться: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документас лицами, поступающими на работу к работодателям — микрофирмам либо индивидуальным предпринимателям; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документас лицами, поступающими на работу к работодателям — физическим лицам, в целях их личного обслуживания и оказания им помощи по ведению домашнего хозяйства (домашними работниками); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документас лицами, поступающими на работу в организации, расположенные в пустынных, высокогорных, малонаселенных районах, если это связано с переездом к месту работы. Перечень таких районов определяется Кабинетом Министров Республики Узбекистан по согласованию с Республиканской трехсторонней комиссией по социально-трудовым вопросам; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадля проведения неотложных работ по предотвращению катастроф, аварий, несчастных случаев, эпидемий, эпизоотий, а также для устранения последствий этих и других чрезвычайных обстоятельств; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документас творческими работниками культурно-зрелищных организаций, организаций телевидения, радиовещания и других средств массовой информации, профессиональными спортсменами, а также иными лицами, участвующими в создании и (или) исполнении (экспонировании) произведений, в соответствии с перечнями работ, профессий, должностей, утверждаемыми Кабинетом Министров Республики Узбекистан по согласованию с Республиканской трехсторонней комиссией по социально-трудовым вопросам; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документас руководителями, заместителями руководителей, главными бухгалтерами организаций и руководителями обособленных подразделений организации, независимо от их организационно-правовых форм и форм собственности, если иное не предусмотрено законом; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документас лицами, получающими образование по очной форме обучения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документас лицами, поступающими на работу по совместительству. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПо соглашению между работником и работодателем срочный трудовой договор может заключаться и в других случаях в соответствии с настоящим Кодексом или иным законом.
