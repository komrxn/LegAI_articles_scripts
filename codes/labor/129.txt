Предварительное испытание при приеме на работу Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаТрудовой договор может быть заключен с предварительным испытанием с целью: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапроверки соответствия работника поручаемой работе; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапринятия работником решения о целесообразности продолжения работы, обусловленной трудовым договором. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПрохождение предварительного испытания должно быть оговорено в трудовом договоре. При отсутствии такой оговорки считается, что работник принят без предварительного испытания. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПредварительное испытание не устанавливается при приеме на работу: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документабеременной женщины, женщины имеющей ребенка в возрасте до трех лет либо отца (опекуна), в одиночку воспитывающего ребенка в возрасте до трех лет; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документалиц из числа социально уязвимых категорий населения, направленных для трудоустройства на резервируемые рабочие места; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документавыпускников высших образовательных организаций, обучавшихся на основе государственных грантов и поступающих на работу по полученной специальности по направлению в течение трех месяцев со дня окончания высшей образовательной организации; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документавыпускников общих средних, средних специальных, профессиональных и высших образовательных организаций, трудоустраивающихся самостоятельно по полученной специальности при поступлении на работу впервые в течение одного года со дня окончания соответствующей образовательной организации; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаработников, с которыми заключается трудовой договор на срок до шести месяцев; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документалиц в возрасте до восемнадцати лет; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документалиц, с которыми работодатель ранее прекратил трудовой договор по отдельным основаниям, в случае их обратного приема на работу; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаучеников, проходивших обучение у работодателя по договору производственного обучения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадругих работников, если это предусмотрено коллективными соглашениями, а также коллективным договором и локальными актами работодателя. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПредварительное испытание может быть установлено только при приеме работника на работу. Не допускается установление предварительного испытания при переводе работника на другую работу и прикомандировании работника к другому работодателю.
