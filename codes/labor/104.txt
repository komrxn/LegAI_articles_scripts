Статья 104 ТК РУз
Содержание трудового договора Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОбязательными для включения в трудовой договор являются следующие условия: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаместо работы — работодатель (организация, ее обособленное подразделение или физическое лицо), у которого работник будет осуществлять трудовую деятельность по обусловленной в договоре трудовой функции, а также местность, в которой работник должен трудиться; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатрудовая функция — работа по определенной профессии, специальности, квалификации либо должности, а также конкретный вид поручаемой работнику работы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадата начала работы — календарная дата, когда работник должен приступить к выполнению работы, обусловленной трудовым договором; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаусловия оплаты труда (в том числе размер тарифной ставки или оклада работника, доплаты, надбавки и поощрительные выплаты); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документасрок трудового договора в случае, когда с работником заключен срочный трудовой договор, а также основания для заключения срочного трудового договора в соответствии с настоящим Кодексом или иными законами; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документарежим рабочего времени и времени отдыха в случае, если для данного работника он отличается от общего режима рабочего времени и времени отдыха, предусмотренного для работников, осуществляющих трудовую деятельность у данного работодателя; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документагарантии и компенсации за работу на условиях, отличающихся от нормальных, если работник принимается на данную работу с указанием характеристик условий труда на рабочем месте; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаусловия, определяющие в необходимых случаях характер работы (подвижной, разъездной, в пути, другой характер работы); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадругие условия в случаях, предусмотренных трудовым законодательством и иными правовыми актами о труде. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли при заключении трудового договора в него не были включены какие-либо условия из числа предусмотренных частью первой настоящей статьи, то это не является основанием для признания трудового договора незаключенным или его прекращения. Трудовой договор должен быть дополнен дополнительными условиями. При этом дополнительные условия определяются дополнительным соглашением к трудовому договору, заключаемым в письменной форме, которое является неотъемлемой частью трудового договора. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ трудовом договоре могут предусматриваться дополнительные условия, не ухудшающие положение работника по сравнению с установленным трудовым законодательством и иными правовыми актами о труде, в частности: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаоб уточнении места работы (с указанием структурного подразделения и его местонахождения) и (или) о рабочем месте; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документао предварительном испытании при приеме на работу; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документао совмещении профессий (должностей); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документао неразглашении государственных секретов и иной охраняемой законом тайны; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаоб обязанности работника отработать после обучения не менее установленного договором срока, если обучение проводилось за счет средств работодателя; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документао видах и об условиях дополнительного страхования работника; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документао социально-бытовых условиях, предоставляемых работнику и членам его семьи; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаоб уточнении применительно к условиям работы работника прав и обязанностей работника и работодателя, установленных трудовым законодательством и иными правовыми актами о труде. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНевключение в трудовой договор каких-либо прав и (или) обязанностей работника и работодателя, установленных трудовым законодательством, не может рассматриваться как отказ от реализации этих прав или исполнения этих обязанностей.
