Сфера действия трудового законодательства Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДействие трудового законодательства распространяется на: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаиндивидуальные трудовые отношения между работником и работодателем; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаобщественные отношения, непосредственно связанные с индивидуальными трудовыми отношениями. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаИндивидуальными трудовыми отношениями являются отношения, основанные на соглашении между работником и работодателем о личном выполнении работником за плату трудовой функции с подчинением работника внутреннему трудовому распорядку при обеспечении работодателем условий труда, предусмотренных трудовым законодательством и иными правовыми актами о труде, трудовым договором. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗапрещается заключение договоров гражданско-правового характера, фактически регулирующих индивидуальные трудовые отношения между работником и работодателем. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК общественным отношениям, непосредственно связанным с индивидуальными трудовыми отношениями, относятся: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаотношения по трудоустройству работника у работодателя; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаотношения по профессиональной подготовке, переподготовке и повышению квалификации работника у работодателя; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаколлективные трудовые отношения, складывающиеся в процессе социального партнерства в сфере труда; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаотношения по контролю и надзору за соблюдением трудового законодательства, правил охраны труда и иных правовых актов о труде; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаотношения по рассмотрению трудовых споров. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПравила, установленные на территории Республики Узбекистан трудовым законодательством, распространяются на трудовые отношения с участием иностранных граждан, лиц без гражданства, организаций, созданных или учрежденных иностранными гражданами, лицами без гражданства либо с их участием, международных организаций и иностранных юридических лиц, если иное не предусмотрено законами или международным договором Республики Узбекистан. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДействие трудового законодательства распространяется на государственных служащих Республики Узбекистан, в том числе на работников, направленных на работу в государственные учреждения Республики Узбекистан за границей (дипломатические, консульские учреждения и иные учреждения), и лиц, проходящих альтернативную службу, с учетом особенностей, предусмотренных законами. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДействие трудового законодательства распространяется на трудовые отношения государственных гражданских служащих в части, не урегулированной Законом Республики Узбекистан «О государственной гражданской службе». Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНа военнослужащих, проходящих военную службу по контракту, а также военнослужащих (сотрудников), проходящих службу в органах внутренних дел и государственной таможенной службы, Службе государственной безопасности, Национальной гвардии Республики Узбекистан и на иных, приравненных к ним лиц, трудовое законодательство распространяется в части, не противоречащей специальному законодательству. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНа судей трудовое законодательство распространяется в части отношений, не урегулированной законодательством о судах. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНа работников органов прокуратуры трудовое законодательство распространяется в части, не противоречащей законодательству, регулирующему деятельность органов прокуратуры. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДействие трудового законодательства распространяется на трудовые отношения спасателей спасательных служб и профессиональных спасательных формирований из числа работающих по трудовому договору в части, не урегулированной Законом Республики Узбекистан «О спасательной службе и статусе спасателя». Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДействие трудового законодательства распространяется на трудовые отношения осужденных по приговору суда за совершение преступлений с учетом изъятий и ограничений, предусмотренных Уголовно-исполнительным кодексом Республики Узбекистан. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаТрудовое законодательство не распространяется на следующих лиц (если в установленном настоящим Кодексом порядке они одновременно не выступают в качестве работодателей или их представителей): Предложения по документуПрослушать аудиоПолучить ссылку из элемента документавоеннослужащих, проходящих срочную военную службу; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документачленов наблюдательных советов организаций; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документачленов ревизионных комиссий (ревизоров), не являющихся работниками данной организации; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документалиц, выполняющих работы (оказывающих услуги) на основании договоров гражданско-правового характера; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадругих лиц, если это установлено законом. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли отношения, связанные с использованием личного труда, возникли на основании договора гражданско-правового характера, но впоследствии в порядке, установленном настоящим Кодексом либо другими законами, были признаны индивидуальными трудовыми отношениями, к таким отношениям применяются положения трудового законодательства и иных правовых актов о труде.
