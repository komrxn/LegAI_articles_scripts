Метод сопоставимой рентабельности Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаМетод сопоставимой рентабельности предусматривает сопоставление операционной рентабельности, сложившейся у лица, являющегося стороной контролируемой сделки, с рыночным интервалом операционной рентабельности в сопоставимых сделках, определенным в порядке, предусмотренном статьей 187 настоящего Кодекса. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаМетод сопоставимой рентабельности может использоваться, в частности, при отсутствии или недостаточности информации, на основании которой можно сделать обоснованный вывод о наличии необходимой степени сопоставимости коммерческих и (или) финансовых условий сопоставляемых сделок и использовать методы, указанные в пунктах 2 и 3 части первой статьи 186 настоящего Кодекса. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДля целей настоящей статьи могут использоваться следующие показатели операционной рентабельности, определяемые в соответствии с частью первой и второй статьи 187 настоящего Кодекса: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) рентабельность продаж; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) рентабельность затрат; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) рентабельность коммерческих и управленческих расходов; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа4) рентабельность активов; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа5) иной показатель рентабельности, отражающий взаимосвязь между осуществляемыми функциями, используемыми активами, принимаемыми экономическими (коммерческими) рисками и уровнем вознаграждения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри выборе конкретного показателя рентабельности учитывается вид деятельности стороны контролируемой сделки, осуществляемые этой стороной функции, используемые активы и принимаемые экономические (коммерческие) риски, полнота, достоверность и сопоставимость данных, используемых для расчета соответствующей рентабельности, а также экономическая обоснованность такого показателя. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДля целей применения настоящей статьи показатели рентабельности используются с учетом следующих особенностей: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) рентабельность продаж используется при последующей перепродаже товаров, приобретенных у взаимосвязанных с перепродавцом лиц, независимым с ним лицам либо при последующей перепродаже товаров, приобретенных у независимых с перепродавцом лиц, взаимосвязанным с ним лицам; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) валовая рентабельность коммерческих и управленческих расходов используется в случаях, указанных в пункте 1 настоящей части, если одновременно выполняются два условия: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаа) перепродавец несет незначительные экономические (коммерческие) риски при приобретении и последующей перепродаже товаров в непродолжительный период времени; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаб) существует прямая взаимосвязь между величиной валовой прибыли перепродавца от продаж и величиной осуществленных им коммерческих и управленческих расходов; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) рентабельность затрат используется при оказании услуг и производстве товаров; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа4) рентабельность активов используется при производстве товаров (в частности, если контролируемые сделки совершаются лицами, осуществляющими капиталоемкую деятельность). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри использовании метода сопоставимой рентабельности с рыночным интервалом рентабельности сопоставляется рентабельность той стороны контролируемой сделки, которая отвечает следующим требованиям: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) сторона контролируемой сделки осуществляет функции, вклад которых в полученную прибыль по сделкам, последовательно совершенным с одним и тем же товаром, меньше, чем вклад другой стороны контролируемой сделки; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) сторона контролируемой сделки принимает меньшие экономические (коммерческие) риски, чем другая сторона контролируемой сделки; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) сторона контролируемой сделки не владеет объектами нематериальных активов, оказывающими существенное влияние на уровень рентабельности. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли сторона контролируемой сделки не отвечает требованиям, предусмотренным пунктами 1 — 3 части шестой настоящей статьи, для сопоставления с рыночным интервалом рентабельности выбирается та сторона контролируемой сделки, которая в наибольшей степени отвечает указанным требованиям. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли рентабельность по контролируемой сделке находится в пределах интервала рентабельности, определенного в порядке, предусмотренном статьей 187 настоящего Кодекса, в целях налогообложения цена контролируемой сделки признается соответствующей рыночным ценам. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли рентабельность по контролируемой сделке меньше минимального значения интервала рентабельности или больше его максимального значения, в целях налогообложения учитывается среднее значение интервала рентабельности. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНа основании среднего значения интервала рентабельности в целях налогообложения осуществляется корректировка прибыли (дохода, выручки) по контролируемой сделке. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПрименение в целях налогообложения корректировки прибыли (дохода, выручки) в соответствии с частями девятой и десятой настоящей статьи производится при условии, что это не приводит к уменьшению суммы налога, подлежащего уплате в бюджетную систему, или к увеличению суммы убытка налогоплательщика.
