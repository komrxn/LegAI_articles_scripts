Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДля исчисления трудового стажа налог в размере не менее одной базовой расчетной величины в год уплачивается в обязательном порядке: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документасубъектами ремесленнической деятельности — членами ассоциации «Хунарманд»; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаучениками школ «Уста — шогирд» — в период их работы до достижения двадцати пяти лет; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаглавой дехканского хозяйства; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документафизическими лицами, состоящими в трудовых отношениях с индивидуальным предпринимателем. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСубъекты ремесленнической деятельности — члены ассоциации «Хунарманд», получающие пенсии и пособия по возрасту, освобождаются от уплаты налога, а для остальных лиц, указанных в частях первой, второй и третьей настоящей статьи, имеющих право на пенсию по возрасту, а также лиц с инвалидностью I и II групп размер налога должен составлять не менее 50 процентов от его установленного минимального размера. Данные льготы предоставляются на основании пенсионного удостоверения или справки медико-социальной экспертной комиссии. При возникновении или прекращении права на льготы в течение календарного года перерасчет налога производится с месяца, в котором возникло или прекращено это право. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаУплата налога производится: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документалицами, указанными в части первой настоящей статьи, — ежемесячно не позднее 15 числа месяца, в котором осуществлялась предпринимательская деятельность; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документалицами, указанными в частях второй и третьей настоящей статьи, — до 31 декабря отчетного года. При этом размер налога исчисляется исходя из размера базовой расчетной величины, установленной на день уплаты. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВновь зарегистрированными индивидуальными предпринимателями и членами семьи, осуществляющими деятельность в форме семейного предпринимательства без образования юридического лица, уплата налога производится с месяца, следующего за месяцем их государственной регистрации в качестве индивидуального предпринимателя. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОбязанность по уплате налога возлагается: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаза членов семьи, осуществляющих деятельность в форме семейного предпринимательства без образования юридического лица, — на члена семьи, зарегистрированного в качестве индивидуального предпринимателя, выступающего от лица субъекта семейного предпринимательства; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаза физических лиц, состоящих в трудовых отношениях с индивидуальным предпринимателем — на индивидуального предпринимателя, с которым у них заключен трудовой договор. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ платежном поручении (приходном ордере) в обязательном порядке должны быть указаны фамилия, имя, отчество, идентификационный номер налогоплательщика и период, за который вносится платеж. При этом по членам семьи — участникам семейного предпринимательства и индивидуальным предпринимателям, нанявшим работника, платежное поручение (приходный ордер) выписывается отдельно на каждого члена семьи и каждого нанятого работника индивидуального предпринимателя. В случае если в платежном поручении (приходном ордере) не указан период, считается, что платеж внесен за месяц (для членов дехканских хозяйств и физических лиц, занятых в дехканском хозяйстве, а также физических лиц, указанных в абзаце пятом части третьей настоящей статьи, — за год), в котором он производится. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПолученная налоговыми органами информация о временном приостановлении деятельности индивидуального предпринимателя является основанием для приостановления начисления налога на тот период, в котором индивидуальный предприниматель не будет осуществлять свою деятельность. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПолученные налоговыми органами карточки учета на каждого наемного работника, выданные в установленном порядке, являются основанием для приостановления начисления налога за каждого работника индивидуального предпринимателя за тот период, в котором индивидуальный предприниматель не будет осуществлять свою деятельность. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае если индивидуальный предприниматель, приостановивший свою деятельность, не сдает в налоговый орган карточки учета на каждого работника индивидуального предпринимателя в установленные сроки, начисление налога по обязательствам индивидуального предпринимателя не приостанавливается. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНалогоплательщики, указанные в части первой, абзаце пятом части третьей настоящей статьи, вправе уплатить налог исходя из суммы полученных доходов в виде оплаты труда, указанных в декларации о совокупном годовом доходе, по налоговым ставкам, указанным в статье 405 настоящего Кодекса, но не менее его минимальных размеров, установленных настоящей статьей, на основании заявления о добровольной уплате налога, подаваемого одновременно с представлением декларации о совокупном годовом доходе. При этом окончательная сумма налога, подлежащая уплате, определяется с учетом сумм, уплаченных в соответствии с частями первой — шестой настоящей статьи.
