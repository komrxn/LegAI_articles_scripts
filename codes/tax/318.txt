Особенности определения налоговой базы некоммерческих организаций Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри соблюдении условий, указанных в статье 58 настоящего Кодекса, следующие доходы некоммерческой организации не подлежат налогообложению: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) целевые поступления на содержание некоммерческих организаций и ведение ими уставной деятельности, указанные в статье 48 настоящего Кодекса; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) превышение суммы положительной курсовой разницы над суммой отрицательной курсовой разницы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) доход от реализации товаров, предназначенных для использования населением по религиозному назначению. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДоходы некоммерческой организации, не указанные в части первой настоящей статьи, подлежат налогообложению в общеустановленном порядке. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗатраты некоммерческой организации, подлежащие отнесению на расходы в соответствии с главой 44 настоящего Кодекса, определяются одним из следующих способов: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) исходя из удельного веса доходов, не указанных в части первой настоящей статьи, в общей сумме доходов некоммерческой организации; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) на основе данных налогового учета, предусматривающего раздельный учет расходов, произведенных за счет доходов, указанных в части первой настоящей статьи, и расходов, произведенных за счет других доходов. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри отсутствии учета, предусмотренного частью третьей настоящей статьи, и (или) использовании доходов, указанных в части первой настоящей статьи, не по целевому назначению (за исключением бюджетных средств), полученные налогоплательщиком целевые средства включаются в состав совокупного дохода этого налогоплательщика и подлежат налогообложению в общеустановленном порядке. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ отношении бюджетных средств, использованных не по целевому назначению, применяются нормы бюджетного законодательства. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПоложения настоящей статьи не распространяются на некоммерческие организации, осуществляющие деятельность в социальной сфере.
