Решение о проведении налогового мониторинга Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗаявление о проведении налогового мониторинга представляется юридическим лицом в налоговый орган по месту учета не позднее 1 июля года, предшествующего периоду, за который будет проводиться налоговый мониторинг. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаФорму заявления о проведении налогового мониторинга утверждает Государственный налоговый комитет Республики Узбекистан. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаС заявлением о проведении налогового мониторинга представляются: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) регламент информационного взаимодействия по установленной форме; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) информация о юридических и физических лицах, которые прямо и (или) косвенно участвуют в юридическом лице, представляющем заявление о проведении налогового мониторинга, и доля такого участия составляет более 25 процентов; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) действующая учетная политика для целей налогообложения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЮридическое лицо, представившее заявление о проведении налогового мониторинга, до принятия налоговым органом решения о его проведении или об отказе в его проведении может отозвать свое заявление на основании письменного заявления об отзыве. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае отзыва заявление о проведении налогового мониторинга не считается поданным. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПо результатам рассмотрения заявления о проведении налогового мониторинга и документов (информации), представленных юридическим лицом в соответствии с частью третьей настоящей статьи, руководитель (заместитель руководителя) налогового органа до 1 ноября года, в котором представлено заявление, принимает одно из следующих решений: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) решение о проведении налогового мониторинга; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) решение об отказе в проведении налогового мониторинга. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРешение об отказе в проведении налогового мониторинга должно быть мотивированным. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОснованиями для принятия решения об отказе в проведении налогового мониторинга являются: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) непредставление или представление не в полном объеме документов (информации) в соответствии с частью третьей настоящей статьи; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) несоблюдение условий, предусмотренных частью пятой статьи 169 настоящего Кодекса; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) несоответствие регламента информационного взаимодействия установленным форме и требованиям. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРешение о проведении налогового мониторинга (решение об отказе в проведении налогового мониторинга) направляется заявителю в течение пяти дней со дня принятия.
