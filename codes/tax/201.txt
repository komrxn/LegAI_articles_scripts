Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДействие соглашения о ценообразовании прекращается досрочно по решению руководителя (заместителя руководителя) Государственного налогового комитета Республики Узбекистан при нарушении налогоплательщиком этого соглашения в течение срока его действия, повлекшем за собой неполную уплату налогов. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСоглашение о ценообразовании может быть также расторгнуто досрочно по соглашению сторон или по решению суда. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРешение Государственного налогового комитета Республики Узбекистан о прекращении соглашения о ценообразовании вручается налогоплательщику (его представителю) под расписку или передается ему иным способом, свидетельствующим о дате его получения, либо направляется налогоплательщику по почте заказным письмом в течение пяти дней со дня принятия соответствующего решения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРешение, направленное налогоплательщику по почте заказным письмом, считается полученным по истечении пяти дней со дня направления заказного письма. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРешение Государственного налогового комитета Республики Узбекистан о прекращении соглашения о ценообразовании может быть обжаловано налогоплательщиком в суде в порядке, установленном законодательством.
