Статья 306 НК РУз
Расходы на амортизацию Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаАмортизируемыми активами в целях настоящего Кодекса признаются основные средства и нематериальные активы, учитываемые налогоплательщиком в качестве таковых в соответствии с настоящим Кодексом. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ целях налогообложения основным средством признается имущество стоимостью более пятидесятикратного размера базовой расчетной величины, установленной в Республике Узбекистан (на дату признания в качестве такового) за единицу (комплект), которое находится у налогоплательщика на правах собственности (если иное не предусмотрено настоящей статьей) и используется им в производстве товаров (при оказании услуг) или для административно-управленческих нужд в течение длительного времени (продолжительностью свыше двенадцати месяцев). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНалогоплательщик вправе устанавливать в учетной политике в целях налогообложения меньший предел стоимости имущества для его учета в составе основных средств. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаАмортизируемым активом также признаются капитальные вложения в предоставленные в аренду и (или) безвозмездное пользование амортизируемые активы в форме неотделимых улучшений, произведенных арендатором (ссудополучателем) с согласия арендодателя (ссудодателя). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаАмортизируемый актив, полученный по договору финансовой аренды (договору лизинга), включается в состав амортизируемых активов у получателя объекта аренды (лизинга). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНе подлежат амортизации следующие активы: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) земля и иные объекты природопользования (вода, недра и другие природные ресурсы); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) продуктивный скот; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) информационно-библиотечный фонд; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа4) музейные предметы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа5) объекты материального культурного наследия; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа6) автомобильные дороги, тротуары, бульвары, скверы общего пользования, сооружения благоустройства, находящиеся в ведении органов государственной власти на местах; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа7) амортизируемый актив, стоимость которого у этого налогоплательщика в целях налогообложения ранее полностью отнесена на расходы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа8) капитальные вложения, не переведенные налогоплательщиком в состав амортизируемых активов; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа9) амортизируемые активы бюджетных организаций, за исключением амортизируемых активов, приобретенных в связи с осуществлением предпринимательской деятельности и используемых для осуществления такой деятельности; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа10) амортизируемые активы (по соответствующей части), приобретенные (созданные) за счет бюджетных средств целевого финансирования. Указанная норма не применяется в отношении амортизируемого актива, полученного налогоплательщиком при приватизации; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа11) амортизируемый актив (по соответствующей части), полученный или приобретенный (созданный) за счет средств, поступивших в соответствии с пунктами 8, 9, 14 и 15 статьи 304 настоящего Кодекса, а также полученный или приобретенный (созданный) за счет средств, высвобожденных в результате предоставленных налоговых и таможенных льгот. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаИз состава амортизируемых активов в целях настоящей статьи исключаются: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) амортизируемый актив, переведенный на консервацию в порядке, установленном законодательством, — на период консервации; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) амортизируемый актив, находящийся по решению налогоплательщика свыше двенадцати месяцев на реконструкции или модернизации, за исключением случаев, когда такой актив в процессе реконструкции или модернизации продолжает использоваться налогоплательщиком в своей деятельности. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ целях настоящей статьи амортизируемый актив принимается на учет по первоначальной стоимости, если иное не предусмотрено настоящей статьей. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПервоначальная стоимость амортизируемого актива, приобретенного (полученного, созданного) после 31 декабря 2021 года, определяется как сумма расходов на его приобретение, сооружение, изготовление, доставку и доведение до состояния, в котором он пригоден для использования, а в случае, если такой актив получен налогоплательщиком безвозмездно либо выявлен в результате инвентаризации, — как рыночная стоимость этого актива, определяемая в соответствии со статьей 299 настоящего Кодекса. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПервоначальная стоимость амортизируемого актива, являющегося предметом финансовой аренды и (или) лизинга, определяется в порядке, аналогичном указанному в части десятой настоящей статьи. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПервоначальной стоимостью амортизируемого актива, полученного в счет вклада акционера (участника, собственника) в уставный фонд (уставный капитал), признается его денежная оценка, согласованная акционерами (участниками, собственниками), но не выше его рыночной стоимости, определяемой в порядке, аналогичном указанному в статье 299 настоящего Кодекса. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПервоначальной стоимостью амортизируемого актива, возведенного (изготовленного, созданного) самим налогоплательщиком, признается сумма фактических затрат по возведению (сооружению, постройке, достройке, изготовлению, созданию) такого актива. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПервоначальная стоимость амортизируемого актива определяется без учета: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаналога на добавленную стоимость, относимого в зачет в соответствии с главой 37 и (или) учитываемого в составе расходов в соответствии со статьей 314 настоящего Кодекса; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документарасходов по процентам и иных расходов, связанных с приобретением амортизируемого актива, подлежащих вычету или учитываемых в составе расходов в соответствии с настоящим Кодексом. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПервоначальной стоимостью амортизируемого актива, переведенного из состава запасов или активов, предназначенных для продажи, является его балансовая стоимость, определенная на дату такого поступления в соответствии с законодательством о бухгалтерском учете. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли амортизируемый актив получен налогоплательщиком вследствие реорганизации (за исключением случаев изменения организационно-правовой формы), первоначальной стоимостью такого актива является его балансовая стоимость, указанная в передаточном акте или разделительном балансе. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПервоначальная (восстановительная) стоимость амортизируемого актива, числящегося у налогоплательщика по данным бухгалтерского учета по состоянию на 1 января 2021 года, в целях налогообложения определяется как его первоначальная стоимость с учетом ранее проведенных переоценок. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри определении восстановительной стоимости амортизируемого актива в целях настоящей статьи учитываются переоценки амортизируемого актива, осуществленные по решению налогоплательщика по состоянию на 1 января 2020 года и отраженные в бухгалтерском учете налогоплательщика после 1 января 2020 года. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДля целей настоящего раздела при проведении налогоплательщиком после 1 января 2021 года переоценки (уценки) стоимости амортизируемого актива положительная (отрицательная) сумма такой переоценки не признается доходом (расходом), не учитывается при определении восстановительной стоимости амортизируемого актива и начислении амортизации. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПервоначальная стоимость амортизируемого актива изменяется в случае осуществления налогоплательщиком капитальных вложений в такой актив при условии их экономической оправданности. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПервоначальная стоимость амортизируемого актива изменяется в случаях достройки, дооборудования, модернизации, реконструкции, технического перевооружения, частичной ликвидации соответствующих объектов и по иным аналогичным основаниям, включая природоохранные цели. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК работам по достройке, дооборудованию, модернизации относятся работы, связанные с изменением технологического или служебного назначения оборудования, здания, сооружения или иного амортизируемого актива и (или) приобретением других новых качеств. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ целях настоящей статьи к реконструкции относится переустройство существующего амортизируемого актива, связанное с совершенствованием производства и повышением его технико-экономических показателей и осуществляемое по проекту реконструкции амортизируемого актива в целях увеличения производственных мощностей, улучшения качества и (или) изменения номенклатуры товаров (услуг). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК техническому перевооружению относится комплекс мероприятий по повышению технико-экономических показателей амортизируемого актива или его отдельных частей на основе изменения техники и технологии, механизации и автоматизации производства, модернизации и замены морально устаревшего и физически изношенного оборудования. Техническое перевооружение включает в себя также создание новых и расширение действующих производств. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаКапитальные вложения в арендованный амортизируемый актив и (или) амортизируемый актив, полученный по договору безвозмездного пользования, амортизируются в следующем порядке: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документакапитальные вложения, стоимость которых возмещается арендатору арендодателем (ссудополучателю ссудодателем), амортизируются арендодателем (ссудодателем); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документакапитальные вложения, произведенные арендатором с согласия арендодателя (ссудополучателем с согласия ссудодателя), стоимость которых не возмещается арендодателем (ссудодателем), амортизируются арендатором (ссудополучателем). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОстаточная стоимость амортизируемого актива, числящегося у налогоплательщика по состоянию на 31 декабря 2020 года, определяется как разница между первоначальной (восстановительной) стоимостью такого амортизируемого актива и суммой начисленной (накопленной) амортизации до 1 января 2021 года. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОстаточная стоимость амортизируемого актива, введенного в эксплуатацию после 31 декабря 2020 года, определяется как разница между его первоначальной стоимостью и суммой начисленной (накопленной) амортизации в целях исчисления налога. В случае аннулирования инвестиционного вычета в соответствии с частью седьмой статьи 308 настоящего Кодекса накопленная амортизация определяется за вычетом инвестиционного вычета. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСтоимость амортизируемого актива относится на расходы через амортизационные отчисления и (или) инвестиционные вычеты, предусмотренные статьей 308 настоящего Кодекса. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСтоимость амортизируемого актива относится на расходы через амортизационные отчисления, начисляемые от первоначальной стоимости, без учета их переоценок, если иное не предусмотрено частью семнадцатой настоящей статьи. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ целях начисления амортизации амортизируемые активы распределяются по амортизационным группам (подгруппам) на дату ввода в эксплуатацию актива в соответствии с положениями настоящей статьи и с учетом статистического классификатора основных фондов Республики Узбекистан. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНачисление амортизации по всем амортизируемым активам (за исключением нематериальных активов) осуществляется линейным методом с применением следующих предельных норм амортизации: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНалогоплательщик вправе в целях исчисления налога начислять амортизацию по нормам ниже установленных настоящей статьей. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСумма амортизации для целей налогообложения определяется налогоплательщиками ежемесячно. Амортизация начисляется отдельно по каждому амортизируемому активу и определяется как произведение его первоначальной стоимости и нормы амортизации, определенной для данного объекта. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНачисление амортизации по амортизируемому активу, в том числе по амортизируемому активу, права на который подлежат государственной регистрации в соответствии с законодательством, начинается с даты, с которой этот актив был введен в эксплуатацию, независимо от даты его государственной регистрации. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли налогоплательщик в течение какого-либо календарного месяца был учрежден, ликвидирован или реорганизован, то амортизация начисляется с учетом следующих особенностей: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) амортизация начисляется учреждаемым, образующимся в результате реорганизации налогоплательщиком с даты его государственной регистрации; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) амортизация начисляется ликвидируемым налогоплательщиком до даты завершения ликвидации, а реорганизуемым налогоплательщиком — до даты завершения реорганизации.
