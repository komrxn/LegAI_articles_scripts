Форма и содержание заявления о выдаче исполнительного листа на принудительное исполнение медиативного соглашения Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗаявление о выдаче исполнительного листа на принудительное исполнение медиативного соглашения подается в письменной форме и должно быть подписано заявителем, заключившим медиативное соглашение, или его представителем. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ заявлении о выдаче исполнительного листа на принудительное исполнение медиативного соглашения должны быть указаны: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) наименование экономического суда, в который подается заявление; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) наименование (фамилия, имя, отчество) сторон медиативного соглашения, их местонахождение (почтовый адрес) или местожительство; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) требование о выдаче исполнительного листа на принудительное исполнение медиативного соглашения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ заявлении о выдаче исполнительного листа на принудительное исполнение медиативного соглашения могут быть указаны номера телефонов, факсов, адреса электронной почты и иные сведения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК заявлению о выдаче исполнительного листа на принудительное исполнение медиативного соглашения прилагаются: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) оригинал или нотариально удостоверенная копия медиативного соглашения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) документы, подтверждающие уплату государственной пошлины и почтовых расходов в установленном порядке и размере; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) уведомление о вручении или иной документ, подтверждающий направление копии заявления о выдаче исполнительного листа на принудительное исполнение медиативного соглашения другой стороне медиативного соглашения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли заявление о выдаче исполнительного листа на принудительное исполнение медиативного соглашения подано представителем стороны медиативного соглашения, то к заявлению прилагается доверенность или иной документ, подтверждающий его полномочия на подписание заявления. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗаявление о выдаче исполнительного листа на принудительное исполнение медиативного соглашения может быть подано не позднее шести месяцев со дня окончания срока для добровольного исполнения медиативного соглашения. При пропуске указанного срока по причинам, признанным экономическим судом уважительными, пропущенный срок может быть восстановлен. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае отказа в восстановлении пропущенного срока, предусмотренного частью шестой настоящей статьи, в принятии заявления о выдаче исполнительного листа на принудительное исполнение медиативного соглашения отказывается по правилам, предусмотренным статьей 154 настоящего Кодекса. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри несоблюдении требований, предусмотренных в части шестой статьи 37 настоящего Кодекса, частях первой, второй, четвертой и пятой настоящей статьи, а также в случае подачи заявления по истечении шестимесячного срока и отсутствия ходатайства о восстановлении пропущенного срока, заявление о выдаче исполнительного листа на принудительное исполнение медиативного соглашения возвращается заявителю по правилам, предусмотренным статьей 155 настоящего Кодекса.
