Прекращение производства по кассационной жалобе (протесту) Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСуд кассационной инстанции прекращает производство по кассационной жалобе (протесту), если: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) кассационная жалоба (протест) подана на судебный акт, который  в соответствии с настоящим Кодексом не может быть обжалован (опротестован) в кассационном порядке, и была ошибочно принята судом кассационной инстанции к производству; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) кассационная жалоба (протест) подана на судебный акт, который был рассмотрен в кассационном порядке, и ошибочно принята судом кассационной инстанции к производству; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) кассационная жалоба (протест) подана лицом, не имеющим права обжаловать (опротестовать) судебный акт; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа4) после принятия кассационной жалобы к производству от лица, подавшего жалобу, поступило заявление об отказе от жалобы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа5) после принятия кассационного протеста к производству от прокурора, принесшего протест, или вышестоящего прокурора поступило заявление об отзыве протеста; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа6) до принятия судебного акта по кассационной жалобе (протесту) юридическое лицо, являющееся стороной в деле, ликвидировано; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа7) до принятия судебного акта по кассационной жалобе (протесту) гражданин, являющийся стороной в деле, умер, а спорное правоотношение не допускает правопреемства. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае, если в кассационной жалобе (протесте) заявлены новые требования, которые не были предметом рассмотрения в суде первой инстанции, принявшего обжалуемое (опротестуемое) решение, суд кассационной инстанции прекращает производство по кассационной жалобе (протесту) в части этих требований. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаО прекращении производства по кассационной жалобе (протесту) выносится определение, копия которого направляется лицам, участвующим в деле, в порядке, предусмотренном статьей 197 настоящего Кодекса. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНа определение о прекращении производства по кассационной жалобе (протесту) может быть подана ревизионная жалоба (протест). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае прекращения производства по кассационной жалобе (протесту) повторное обращение того же лица по тем же основаниям в суд с кассационной жалобой (протестом) не допускается.
