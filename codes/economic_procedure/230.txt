Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри подготовке дела к судебному разбирательству по ходатайству стороны третейского разбирательства судья может истребовать из третейского суда материалы дела, по которому испрашивается исполнительный лист, по правилам, предусмотренным настоящим Кодексом для истребования доказательств. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСтороны третейского разбирательства извещаются экономическим судом о времени и месте судебного заседания. Неявка указанных лиц, надлежаще извещенных о времени и месте судебного заседания, не является препятствием для рассмотрения дела. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри рассмотрении дела в судебном заседании экономический суд устанавливает наличие или отсутствие предусмотренных статьей 231 настоящего Кодекса оснований для отказа в выдаче исполнительного листа на принудительное исполнение решения третейского суда путем исследования представленных в экономический суд доказательств обоснования заявленных требований и возражений. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри рассмотрении дела в судебном заседании экономический суд не вправе исследовать обстоятельства, установленные третейским судом, либо пересматривать решение третейского суда по существу. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли в производстве экономического суда, указанного в части третьей статьи 37 настоящего Кодекса, находится заявление об отмене решения третейского суда, экономический суд, в котором рассматривается заявление о выдаче исполнительного листа на принудительное исполнение этого решения, по ходатайству должника может отложить рассмотрение заявления о выдаче исполнительного листа на принудительное исполнение решения третейского суда.
