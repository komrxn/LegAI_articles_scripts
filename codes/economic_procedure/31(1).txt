Передача материалов дела из экономического суда в другой суд по подведомственности Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае предъявления требования в экономический суд с нарушением правил подведомственности исковое заявление (заявление) передается на рассмотрение в суд по гражданским делам или административный суд по подведомственности. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли в ходе рассмотрения дела выявлено, что исковое заявление (заявление) было принято к производству с нарушением правил подведомственности, экономический суд выносит определение о передаче материалов дела на рассмотрение суда по гражданским делам или административного суда по подведомственности и прекращении экономического судопроизводства. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОпределение о передаче искового заявления (заявления), материалов дела из экономического суда в другой суд по подведомственности может быть обжаловано (опротестовано). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаИсковое заявление (заявление), материалы дела, переданные из экономического суда в другой суд по подведомственности, должны быть приняты судом, которому они переданы.
