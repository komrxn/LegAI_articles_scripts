Порядок рассмотрения дела судом апелляционной инстанции Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСуд апелляционной инстанции рассматривает дело в судебном заседании по правилам рассмотрения дела судом первой инстанции с учетом особенностей, установленных настоящей главой. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПервое судебное заседание по апелляционной жалобе (протесту) должно быть проведено не позднее двадцати дней со дня вынесения определения о принятии апелляционной жалобы (протеста) к производству. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ суде апелляционной инстанции не применяются правила: об объединении дел; об изменении предмета или основания иска, размера исковых требований; о предъявлении встречного иска; о привлечении к участию в деле третьих лиц, а также иные правила, установленные настоящим Кодексом только для рассмотрения дела в суде первой инстанции, за исключением случая рассмотрения дела по правилам производства в суде первой инстанции. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНеявка в судебное заседание суда апелляционной инстанции лица, подавшего апелляционную жалобу (протест), и других лиц, участвующих в деле, надлежаще извещенных о времени и месте проведения судебного разбирательства, не препятствует рассмотрению дела в их отсутствие. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРассмотрение дела в суде апелляционной инстанции начинается докладом председательствующего или одного из судей. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДокладчик излагает обстоятельства дела, содержание решения суда первой инстанции, доводы апелляционной жалобы (протеста) и поступивший на нее отзыв. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПосле доклада суд заслушивает объяснения явившихся в судебное заседание лиц, участвующих в деле, которые вправе приводить также доводы, не указанные в апелляционной жалобе (протесте), и представлять дополнительные доказательства. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПервым выступает лицо, подавшее апелляционную жалобу, или его представитель, либо прокурор, если дело рассматривается по его протесту. В случае обжалования решения обеими сторонами, первым выступает истец. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаГосударственные органы и иные лица, участвующие в деле в порядке, предусмотренном статьей 50 настоящего Кодекса, если ими не обжаловано решение, выступают после сторон и третьих лиц. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПосле заслушивания объяснений лиц, участвующих в деле, суд апелляционной инстанции удаляется в совещательную комнату для принятия постановления. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПо окончании судебного разбирательства объявляется резолютивная часть принятого постановления, которая приобщается к делу. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПостановление в полном объеме составляется и подписывается в течение пяти дней. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПредседательствующий после объявления резолютивной части постановления разъясняет лицам, участвующим в деле, срок составления постановления в полном объеме и его направления им, а также порядок обжалования (опротестования) постановления.
