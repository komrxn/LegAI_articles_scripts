Рассмотрение вопроса об утверждении мирового соглашения Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВопрос об утверждении мирового соглашения рассматривается в судебном заседании судом, в производстве которого находится дело. Лица, участвующие в деле, извещаются о времени и месте судебного заседания в порядке, предусмотренном статьей 127 настоящего Кодекса. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНеявка лиц, надлежаще извещенных о времени и месте судебного заседания, не препятствует рассмотрению вопроса об утверждении мирового соглашения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОб утверждении мирового соглашения суд выносит определение, в котором указывается о прекращении производства по делу. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае заключения мирового соглашения в стадии исполнения судебного акта, оно представляется для утверждения в суд первой инстанции, рассмотревший дело. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВопрос утверждения мирового соглашения, заключенного в стадии исполнения, рассматривается в пятнадцатидневный срок со дня его представления в суд по правилам, предусмотренным в частях первой и второй настоящей статьи. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРезолютивная часть определения об утверждении мирового соглашения должна содержать: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) условия мирового соглашения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) вывод об утверждении мирового соглашения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) распределение судебных расходов; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа4) сведение о прекращении производства по делу. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРезолютивная часть постановления суда апелляционной или кассационной либо ревизионной инстанции об утверждении мирового соглашения, кроме сведений, предусмотренных в части шестой настоящей статьи, должно содержать указание об отмене всех ранее принятых по делу судебных актов. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРезолютивная часть определения суда об утверждении мирового соглашения, заключенного в стадии исполнения, должна содержать условия мирового соглашения и вывод об утверждении мирового соглашения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСудебные акты об утверждении мирового соглашения могут быть обжалованы (опротестованы). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаМировое соглашение исполняется лицами, его заключившими, добровольно в порядке и в сроки, которые предусмотрены этим соглашением. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаМировое соглашение, утвержденное судом, не исполненное добровольно в сроки, которые предусмотрены этим соглашением, подлежит принудительному исполнению по правилам раздела V настоящего Кодекса на основании исполнительного листа, выдаваемого судом по ходатайству лица, заключившего мировое соглашение, а в части взыскания судебных расходов — по инициативе суда.
