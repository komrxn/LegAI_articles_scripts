Отложение судебного разбирательства Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСуд вправе отложить судебное разбирательство при невозможности рассмотрения дела в судебном заседании в случае непредвиденных и чрезвычайных обстоятельств, в том числе: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапо ходатайству стороны, в случае ее обращения за содействием к суду в целях мирного урегулирования спора; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаесли лицо, участвующее в деле, и надлежаще извещенное о времени и месте судебного заседания, заявило ходатайство об отложении судебного разбирательства с обоснованием причины неявки в судебное заседание, в случае если рассмотрение дела без участия этого лица невозможно; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапо ходатайству лица, участвующего в деле, в связи с неявкой его представителя в судебное заседание по уважительной причине; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав случае неявки кого-либо из участников судебного процесса, если суд считает невозможным рассмотрение дела без участия этого лица; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапо ходатайству стороны об отложении судебного разбирательства в связи с необходимостью представления или истребования у третьих лиц дополнительных доказательств; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав случае невозможности установления видеоконференцсвязи при проведении судебного заседания в режиме видеоконференцсвязи; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав случае неявки кого-либо из участников судебного процесса и отсутствия доказательств надлежащего извещения указанного лица о времени и месте судебного разбирательства; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав случае необходимости замены ненадлежащего ответчика надлежащим ответчиком; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав случае необходимости замены выбывшей стороны в спорном правоотношении ее правопреемником; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав случае необходимости привлечения к участию в деле второго ответчика или третьей стороны. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСудебное разбирательство может быть отложено на срок, необходимый для устранения обстоятельств, послуживших основанием для отложения, в пределах срока судебного разбирательства, предусмотренного настоящим Кодексом, но не более десяти дней. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаКоличество отложений судебного разбирательства не может превышать трех раз. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОб отложении судебного разбирательства выносится определение. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли в судебном заседании присутствуют стороны, суд до отложения судебного разбирательства вправе допросить явившихся свидетелей. Показания этих свидетелей оглашаются в новом судебном заседании. Повторный вызов этих свидетелей в новое судебное заседание производится только в случаях необходимости. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаО времени и месте нового заседания суда лица, участвующие в деле, но не явившиеся в судебное заседание, извещаются в порядке, предусмотренном статьей 127 настоящего Кодекса. Лица, участвующие в деле, явившиеся в судебное заседание, извещаются под расписку, приобщаемую к материалам дела. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПосле отложения судебное разбирательство продолжается. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае необходимости разбирательство дела после его отложения начинается сначала.
