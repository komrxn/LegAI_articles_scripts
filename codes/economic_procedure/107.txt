Основания оставления искового заявления без рассмотрения Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСуд оставляет исковое заявление без рассмотрения, если: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) в производстве суда по гражданским делам, экономического суда, третейского суда, арбитражного суда имеется дело по спору между теми же лицами, о том же предмете и по тем же основаниям; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) имеется соглашение лиц, участвующих в деле, о передаче данного спора на разрешение третейского суда или арбитража и возможность обращения к третейскому суду или арбитражу не утрачена и если ответчик, возражающий против рассмотрения дела в экономическом суде, не позднее своего первого заявления по существу спора заявит ходатайство о передаче спора на разрешение третейского суда или арбитража; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) исковое заявление не подписано или подписано лицом, не имеющим права подписывать его, либо лицом, должностное положение или фамилия и инициалы которого не указаны; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа4) истец не обращался в банк или иную кредитную организацию за получением с ответчика задолженности, когда она согласно законодательству или договору должна быть получена через банк либо иную кредитную организацию; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа5) истцом не соблюден досудебный (претензионный) порядок урегулирования спора с ответчиком, когда это предусмотрено законом для данной категории споров или договором; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа51) истцом не соблюден порядок урегулирования спора с проведением процедуры медиации с ответчиком, когда это предусмотрено законом для данной категории споров или договором; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа52) стороны, заявившие ходатайство о проведении процедуры медиации, по истечении срока ее проведения не явились на заседание суда без уважительных причин; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа53) между сторонами заключено медиативное соглашение; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа6) истец не явился на первое заседание суда и не заявил о рассмотрении дела без его участия; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа7) при рассмотрении заявления об установлении фактов, имеющих юридическое значение, выясняется, что возник спор о праве; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа8) истец, в интересах которого прокурором, государственным органом и иным лицом предъявлен иск, отказался от иска; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа9) в отношении ответчика возбуждено дело о неплатежеспособности и предъявленное к ответчику требование согласно закону подлежит рассмотрению в рамках дела о неплатежеспособности.
