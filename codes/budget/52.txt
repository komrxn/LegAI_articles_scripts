Формирование доходов бюджета Республики Каракалпакстан, местных бюджетов областей и города Ташкента Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДоходы бюджета Республики Каракалпакстан, местных бюджетов областей и города Ташкента формируются за счет: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) в соответствии с установленными нормативами: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаналога на прибыль; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаналога на доходы физических лиц; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаакцизного налога; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаналога за пользование недрами (за исключением налога за пользование недрами по строительным материалам); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) налога за пользование водными ресурсами; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) налога на имущество; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа4) земельного налога; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа5) налога за пользование недрами по строительным материалам; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа6) налога с оборота; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа7) безвозмездных денежных поступлений от юридических и физических лиц, а также иностранных государств; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа8) сбора за приобретение и (или) временный ввоз на территорию Республики Узбекистан автотранспортных средств; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа9) прочих доходов. К прочим доходам относятся поступления от реализации имущества, обращенного в доход государства, доходы от размещения, предоставления в пользование и продажи государственных активов по установленным нормативам, государственные пошлины, сборы, возмещения и штрафные санкции, поступления от реализации бесхозного имущества, имущества, перешедшего по праву наследования к государству, кладов, по праву подлежащих обращению в доход государства, дивиденды (доходы) по доле органов государственной власти на местах и другие доходы, направляемые в бюджет Республики Каракалпакстан, местные бюджеты областей и города Ташкента в соответствии с законодательством. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаМежбюджетные трансферты, выделяемые из республиканского бюджета Республики Узбекистан, считаются доходами республиканского бюджета Республики Каракалпакстан, областных бюджетов областей и городского бюджета города Ташкента. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОтдельные налоги и другие доходы, поступающие в республиканский бюджет Республики Узбекистан, могут распределяться в республиканский бюджет Республики Каракалпакстан, областные бюджеты областей, городской бюджет города Ташкента в размерах и порядке, установленных законодательством. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаНалоги и другие доходы в соответствии с установленными нормативами могут перераспределяться между республиканским бюджетом Республики Каракалпакстан, областными бюджетами областей, городским бюджетом города Ташкента и бюджетами районов и городов.
