Государственный финансовый контроль, осуществляемый Инспекцией государственного финансового контроля Департамента по борьбе с экономическими преступлениями при Генеральной прокуратуре Республики Узбекистан Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаИнспекция государственного финансового контроля Департамента по борьбе с экономическими преступлениями при Генеральной прокуратуре Республики Узбекистан осуществляет государственный финансовый контроль за: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документасоблюдением бюджетного законодательства, бюджетной и сметно-штатной дисциплины; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаисполнением расходов бюджетов бюджетной системы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаочередностью, соблюдением сроков и полнотой осуществления платежей за счет средств бюджетов бюджетной системы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документацелевым расходованием средств бюджетов бюджетной системы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документацелевым использованием средств из централизованных источников, выделяемых для осуществления государственных программ развития; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаправильным и целевым использованием средств в иностранной валюте дипломатическими учреждениями Республики Узбекистан, находящимися за рубежом; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаправильностью назначения, начисления и своевременностью выплаты пенсий и пособий; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документацелевым использованием бюджетных субсидий, ссуд и кредитных линий; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документацелевым использованием кредитов (займов), привлеченных Республикой Узбекистан или под ее гарантию; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документацелевым использованием гуманитарной помощи и средств технического содействия; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаправильностью проведения тендерных (конкурсных) торгов по государственным закупкам.
