Роспись доходов и расходов Государственного бюджета и бюджетов государственных целевых фондов Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРоспись доходов и расходов Государственного бюджета и бюджетов государственных целевых фондов (далее — роспись доходов и расходов) составляется в целях обеспечения: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документасоответствия прогноза доходов и расходов, составленных в соответствии с бюджетной классификацией, утвержденным их параметрам; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документамониторинга исполнения параметров доходов и расходов. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДоходная часть росписи доходов и расходов состоит из годовых объемов доходов соответствующего бюджета с помесячной разбивкой в соответствии с бюджетной классификацией. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРасходная часть росписи доходов и расходов состоит из годовых объемов расходов соответствующего бюджета с помесячной разбивкой, составляемой на основе утвержденных сводных смет расходов смет расходов на финансовый год в соответствии с бюджетной классификацией. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаИтоговые данные росписи доходов и расходов, за исключением межбюджетных трансфертов, должны соответствовать суммам, принятым в утвержденном годовом бюджете. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРоспись доходов и расходов составляется: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаМинистерством финансов Республики Узбекистан — по республиканскому бюджету Республики Узбекистан; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаМинистерством финансов Республики Каракалпакстан, финансовыми управлениями хокимиятов областей и города Ташкента — по республиканскому бюджету Республики Каракалпакстан, областным бюджетам областей и городскому бюджету города Ташкента; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документафинансовыми отделами районных и городских хокимиятов — по бюджетам районов и городов; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаорганами — распорядителями государственных целевых фондов — по государственным целевым фондам.
