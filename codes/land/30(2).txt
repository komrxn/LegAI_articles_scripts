Статья 30(2) ЗК РУз
Права и обязанности участников сервитута Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаУчастниками сервитута являются землевладельцы, землепользователи, арендаторы и собственники земельных участков и лицо, в интересах которого установлен сервитут. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗемлевладельцы, землепользователи, арендаторы и собственники земельных участков имеют право: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатребовать от лица, в интересах которого установлен сервитут, целевое использование сервитута; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатребовать от лица, в интересах которого установлен сервитут, своевременную плату в случаях, когда сервитут установлен за плату; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатребовать приведения земельного участка в состояние, пригодное для использования его в соответствии с основным видом разрешенного использования по истечении срока сервитута; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатребовать расторжения договора, если сервитут не используется лицом, в интересах которого установлен сервитут, в целях, установленных договором; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатребовать расторжения договора в случае систематической неуплаты лицом, в интересах которого установлен сервитут, установленной платы за сервитут (при наличии), а также при появлении у него возможности проезда на свою территорию через другой земельный участок, предназначенный для общественных нужд. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗемлевладельцы, землепользователи, арендаторы и собственники земельных участков обязаны: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапровести государственную регистрацию договора о сервитуте совместно с лицом, в интересах которого установлен сервитут; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документас момента государственной регистрации договора о сервитуте, предоставить лицу, в интересах которого установлен сервитут, возможность ограниченного пользования земельным участком; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документане препятствовать лицу, в интересах которого установлен сервитут, использовать сервитут по назначению; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапривести земельный участок в порядок, охранять, ремонтировать и иным образом обслуживать своими силами и за свой счет с учетом основного вида разрешенного использования земельного участка для поддержания его в надлежащем состоянии. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЛицо, в интересах которого установлен сервитут, имеет право: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаограниченного пользования земельным участком для целей, указанных в договоре о сервитуте; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатребовать расторжения договора о сервитуте в случае существенного нарушения условий договора о сервитуте; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаобращения в суд в установленном порядке с требованием об устранении препятствий, возникших при использовании сервитута, и о возмещении причиненного ущерба. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЛицо, в интересах которого установлен сервитут, может иметь и другие права в соответствии с договором о сервитуте и законодательством. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЛицо, в интересах которого установлен сервитут, обязано: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапровести государственную регистрацию договора о сервитуте совместно с землевладельцами, землепользователями, арендаторами и собственниками земельных участков; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаиспользовать сервитут только в целях, определенных договором о сервитуте; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документасвоевременно вносить плату за сервитут (при установлении платы); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапо первому требованию землевладельцев, землепользователей, арендаторов и собственников земельных участков прекращать ограниченное пользование чужим земельным участком в случае возникновения возможности проезда на его территорию через другой земельный участок, предназначенный для общественных нужд; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документав случае получения уведомления от землевладельцев, землепользователей, арендаторов и собственников земельных участков о заключении договора о сервитуте в отношении земельного участка с третьими лицами отвечать в письменной форме с момента получения такого уведомления.
