Внутрихозяйственное распределение земельных участков. Семейный (коллективный) подряд Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ сельскохозяйственных кооперативах и других сельскохозяйственных предприятиях, учреждениях и организациях земельные участки, как правило, передаются в пользование на срок не менее пяти лет семьям на условиях договора семейного подряда для производства сельскохозяйственной продукции. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗемельные участки для выполнения определенных видов сельскохозяйственных работ могут также передаваться в пользование на срок не менее пяти лет группам членов сельскохозяйственных кооперативов  и работников других сельскохозяйственных предприятий, учреждений и организаций на условиях договора коллективного подряда. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаСемейный подряд — форма внутрихозяйственной организации производства и труда, основанная на непосредственном участии членов семьи в производстве сельскохозяйственной продукции на земельных участках, предоставленных сельскохозяйственным кооперативом и другим сельскохозяйственным предприятием, учреждением и организацией семье во временное пользование на условиях договора семейного подряда. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДоговор семейного (коллективного) подряда заключается ежегодно между сельскохозяйственным кооперативом, другим коллективным сельскохозяйственным предприятием в лице их органов управления или работодателем (администрацией) иных сельскохозяйственных предприятий, учреждений и организаций и главой семьи (уполномоченным представителем коллектива). Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ договоре семейного (коллективного) подряда предусматриваются: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаразмер, местоположение и состояние земельного участка; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаусловия использования земельного участка, обязательства сторон по повышению урожайности и качества земли в соответствии со схемой севооборота, по сохранению и повышению плодородия почвы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаколичество и виды производимой сельскохозяйственной продукции, ее качества; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документацена, условия оплаты и реализации произведенной сельскохозяйственной продукции; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаобеспечение подрядчика оросительной водой и материально-техническими ресурсами; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаответственность сторон за невыполнение договорных обязательств и другие условия по усмотрению сторон. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗемельные участки, предоставляемые на условиях семейного (коллективного) подряда, используются строго в соответствии с основными видами разрешенного использования, при этом не допускается уменьшение размеров площади пахотных земель. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРазмеры земельного участка и его границы могут быть изменены по соглашению сторон. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОплата труда при семейном (коллективном) подряде осуществляется по конечным результатам, предусмотренным договором, количеству, качеству и цене произведенной продукции. Наряду с этим члены семьи (подрядчики) — владельцы паев получают дивиденды, которые определяются по конечным результатам деятельности хозяйства за год. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПлата за землю с земельного участка, предоставленного на условиях семейного (коллективного) подряда, с подрядчика не взимается. Земельный налог с этих участков уплачивается в установленном порядке сельскохозяйственным кооперативом и другими сельскохозяйственными предприятиями, учреждениями и организациями. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПередача земельного участка, полученного на условиях договора подряда, в аренду или субподряд запрещается. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае изъятия земельного участка для общественных нужд подрядчик имеет право на возмещение убытков и других затрат в соответствии со статьей 86 настоящего Кодекса.
