Права и обязанности перевозчика Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПеревозчик вправе не принимать к перевозке товары и (или) транспортные средства в случаях, если: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатоваросопроводительные документы оформлены с нарушением установленного порядка; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документасредства таможенной идентификации, наложенные на транспортное средство и упаковку товаров, не исключают возможности доступа к перевозимым товарам без нарушения таких обеспечений. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПеревозчик может иметь и иные права в соответствии с законодательством. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри перевозке по таможенной территории товаров, находящихся под таможенным контролем, перевозчик обязан: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаобеспечить сохранность товаров, пломб и печатей либо иных средств таможенной идентификации, если они использовались; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документане допускать проведение грузовых операций с товарами без согласия таможенных органов, за исключением перегрузки товара на другое транспортное средство в случае, предусмотренном частью первой статьи 225 настоящего Кодекса; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапоместить перевозимые товары в зону таможенного контроля; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаподдерживать транспортные средства в надлежащем техническом состоянии и обеспечивать их соответствие требованиям к оборудованию транспортных средств для перевозок товаров под таможенным контролем в соответствии со статьей 227 настоящего Кодекса; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадоставлять товары и (или) транспортные средства в неизменном состоянии, кроме изменений вследствие естественного износа либо убыли при нормальных условиях транспортировки и хранения, без использования в каких-либо иных целях; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документауплачивать таможенные платежи при утрате товаров либо их передаче другим лицам без согласия таможенного органа. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПосле прибытия в место доставки перевозчик не имеет права без письменного уведомления таможенного органа оставлять товары и (или) транспортное средство без присмотра на месте стоянки, изменять место стоянки, высаживать пассажиров, производить какие-либо грузовые и упаковочные операции с товарами, изменять, удалять или уничтожать средства таможенной идентификации. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПеревозчик несет ответственность перед таможенным органом за обеспечение включения информации о всех товарах в товаросопроводительные документы или за доведение информации о товарах до сведения таможенного органа. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПеревозчик может нести и иные обязанности в соответствии с законодательством.
