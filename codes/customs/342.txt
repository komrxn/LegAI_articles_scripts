Применение залога товаров в качестве обеспечения уплаты таможенных платежей Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ качестве обеспечения уплаты таможенных платежей может применяться залог товаров на основании договора о залоге. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПредметом залога могут быть любые товары, за исключением: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаэлектрической, тепловой и иных видов энергии; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапредприятий, зданий, строений, сооружений, земельных участков, космических объектов; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатоваров, уже заложенных для обеспечения иного обязательства; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаобъектов интеллектуальной собственности; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатоваров, запрещенных к ввозу на таможенную территорию; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаскоропортящихся товаров; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаживотных; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатоваров, находящихся за пределами Республики Узбекистан; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатоваров, возможность реализации которых ограничена; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатоваров, постоянный контроль за местонахождением и использованием которых таможенный орган не может обеспечить; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапродукции и отходов производства, свободная реализация которых запрещена в соответствии с законодательством. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаТовары, перемещаемые через таможенную границу и подлежащие сертификации или контролю другими государственными органами, могут быть предметом залога только при наличии сертификатов, разрешений и иных документов, необходимых при выпуске товаров. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДля определения рыночной стоимости предмета залога осуществляется его оценка за счет залогодателя в соответствии с требованиями законодательства об оценочной деятельности. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРыночная стоимость предмета залога не может быть ниже размера обязательства по таможенным платежам, уплата которых обеспечивается предметом залога, включая пятнадцать процентов от рыночной стоимости товаров на покрытие необходимых расходов по его хранению и реализации. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗалог товаров не применяется в качестве обеспечения уплаты таможенных платежей в случаях, предусмотренных в статьях 93 и 236 настоящего Кодекса, а также при лицензировании деятельности таможенного склада, магазина беспошлинной торговли и свободного склада.
