Признание задолженности по уплате таможенных платежей безнадежной Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЗадолженность по уплате таможенных платежей, которая не может быть погашена вследствие прекращения обязательства по решению суда либо вследствие неплатежеспособности, ликвидации или смерти должника, считается безнадежной задолженностью. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПо решению таможенных органов задолженность по уплате таможенных платежей признается безнадежной в следующих случаях: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа1) ликвидации юридического лица — в части его задолженности по уплате таможенных платежей, не погашенной по причине недостаточности имущества юридического лица и (или) невозможности ее погашения учредителями (участниками) этого юридического лица в пределах и порядке, установленных законодательством; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа2) признания банкротом индивидуального предпринимателя — в части задолженности по уплате таможенных платежей, не погашенной по причине недостаточности имущества должника; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа3) смерти физического лица или объявления физического лица умершим либо признания физического лица банкротом — в части его задолженности по уплате таможенных платежей, не погашенной по причине недостаточности его имущества, в том числе в случае перехода наследства в собственность государства; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документа4) снятия с учета в налоговом органе иностранного юридического лица — в части его задолженности по уплате таможенных платежей, не погашенной по причине недостаточности имущества постоянного учреждения и невозможности ее погашения со стороны юридического лица — нерезидента Республики Узбекистан в пределах и порядке, установленных законодательством. Указанная задолженность, признанная безнадежной к взысканию, подлежит восстановлению при повторной постановке на учет этого иностранного юридического лица в налоговом органе.
