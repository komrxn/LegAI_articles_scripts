Личный досмотр Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЛичный досмотр проводится в отношении физического лица, следующего через таможенную границу и находящегося в зоне таможенного контроля или транзитной зоне аэропорта, открытого для международного сообщения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЛичный досмотр проводится должностным лицом таможенного органа при наличии оснований полагать, что физическое лицо скрывает при себе и добровольно не предъявляет товары, запрещенные к ввозу на таможенную территорию или вывозу с этой территории, либо товары, перемещаемые с нарушением порядка, установленного настоящим Кодексом. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаРешение о проведении личного досмотра принимается начальником таможенного органа или лицом, его замещающим, в письменной форме. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПеред началом личного досмотра должностное лицо таможенного органа обязано: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаобъявить физическому лицу решение о проведении личного досмотра; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаознакомить физическое лицо с его правами при проведении личного досмотра; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапредложить добровольно выдать скрываемые при себе товары. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаВ случае отказа физического лица от личного досмотра, в решении о проведении личного досмотра делается отметка, удостоверяемая подписью должностного лица таможенного органа, объявившего решение о проведении личного досмотра. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли досматриваемое лицо не выполняет или оказывает сопротивление выполнению законных требований должностного лица таможенного органа, эти действия являются основанием для административного задержания указанного лица в соответствии с законодательством об административной ответственности. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЛичный досмотр проводится должностным лицом таможенного органа одного пола с досматриваемым лицом в присутствии двух понятых того же пола в изолированном помещении, отвечающем санитарно-гигиеническим требованиям. Доступ в это помещение других физических лиц и возможность наблюдения за проведением личного досмотра с их стороны не допускается. Обследование тела, досматриваемого лица, должно проводиться только медицинскими работниками с использованием, при необходимости, специальной медицинской техники. При необходимости при личном досмотре привлекается переводчик. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри личном досмотре несовершеннолетнего или недееспособного лица вправе присутствовать его законные представители (родители, усыновители, опекуны, попечители) или лица, его сопровождающие. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЛичный досмотр проводится в пределах, необходимых для обнаружения скрытых досматриваемым лицом при себе товаров, и форме, исключающей унижение чести и достоинства человека, причинение вреда его здоровью и имуществу. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДосматриваемое лицо или его законный представитель либо лицо, его сопровождающее, в ходе личного досмотра имеет право: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документатребовать объявления решения начальника таможенного органа или лица, его замещающего, о проведении личного досмотра; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапользоваться родным языком, а также пользоваться услугами переводчика; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаознакомиться со своими правами и обязанностями; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадавать объяснения, заявлять ходатайства; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документазнакомиться с протоколом личного досмотра по окончании его составления и делать заявления, подлежащие внесению в протокол; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаобжаловать в установленном порядке решение таможенного органа, действия (бездействие) его должностных лиц. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДосматриваемое лицо или его законный представитель либо лицо, его сопровождающее, может иметь и иные права в соответствии с законодательством. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДосматриваемое лицо или его законный представитель либо лицо, сопровождающее досматриваемое лицо в ходе личного досмотра, обязано выполнять законные требования должностного лица таможенного органа. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПо результатам личного досмотра составляется протокол по форме, устанавливаемой Государственным таможенным комитетом Республики Узбекистан. Протокол подписывается: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадолжностным лицом таможенного органа, проводившим личный досмотр; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапонятыми; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документафизическим лицом, в отношении которого был проведен личный досмотр, либо его законным представителем или лицом, его сопровождающим; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документамедицинским работником — при обследовании им органов тела, досматриваемого лица; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаиными лицами, участвовавшими при личном досмотре. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаФизическому лицу, в отношении которого был проведен личный досмотр, под роспись предоставляется копия протокола о проведении личного досмотра.
