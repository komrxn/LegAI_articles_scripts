Ограничительные условия договора комплексной предпринимательской лицензии Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДоговором комплексной предпринимательской лицензии могут быть предусмотрены ограничительные (эксклюзивные) условия, в частности: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаобязательство лицензиара не выдавать других аналогичных комплексных предпринимательских лицензий для их использования на закрепленной за лицензиатом территории либо воздержаться от непосредственной самостоятельной деятельности на этой территории; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаобязательство лицензиата не конкурировать с лицензиаром на территории использования комплексной предпринимательской лицензии; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаотказ лицензиата от получения других комплексных предпринимательских лицензий у конкурентов (потенциальных конкурентов) лицензиара; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаобязательство лицензиата согласовывать с лицензиаром место расположения коммерческих помещений, используемых при осуществлении предоставленных по договору исключительных прав, а также их внешнее и внутреннее оформление. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаОграничительные условия могут быть признаны недействительными по иску антимонопольного органа или иного заинтересованного лица, если они, с учетом состояния соответствующего рынка и экономического положения сторон, противоречат законодательству о конкуренции. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЯвляются недействительными ограничительные условия договора комплексной предпринимательской лицензии, в силу которых: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документалицензиар вправе определять цену продажи товара лицензиатом или цену работ (услуг), выполняемых (оказываемых) лицензиатом, либо устанавливать верхний или нижний предел указанных цен; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документалицензиат вправе продавать товары, выполнять работы или оказывать услуги исключительно определенной категории покупателей (заказчиков) либо исключительно покупателям (заказчикам), имеющим место нахождения (место жительства) на определенной в договоре территории.
