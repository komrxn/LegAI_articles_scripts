Право, применяемое к договору при отсутствии соглашения сторон Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри отсутствии соглашения сторон договора о подлежащем применению праве к этому договору применяется право страны, где учреждена, имеет место жительства или основное место деятельности сторона, являющаяся: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапродавцом — в договоре купли-продажи; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадарителем — в договоре дарения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаарендодателем или наймодателем — в договоре аренды (имущественного найма); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документассудодателем — в договоре безвозмездного пользования имуществом; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаподрядчиком — в договоре подряда; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаперевозчиком — в договоре перевозки; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаэкспедитором — в договоре транспортной экспедиции; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документакредитором — в договоре займа или ином кредитном договоре; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаповеренным — в договоре поручения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документакомиссионером — в договоре комиссии; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документахранителем — в договоре хранения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документастраховщиком — в договоре страхования; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапоручителем — в договоре поручительства; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документазалогодателем — в договоре залога; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаклиентом — в договоре финансирования под уступку денежного требования (факторинга); Предложения по документуПрослушать аудиоПолучить ссылку из элемента документалицензиаром — в лицензионном договоре о пользовании исключительными правами. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри отсутствии соглашения сторон договора о подлежащем применению праве применяется независимо от положений части первой настоящей статьи: Предложения по документуПрослушать аудиоПолучить ссылку из элемента документак договору о недвижимом имуществе — право страны, где это имущество находится; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документак договору о совместной деятельности и договору строительного подряда — право страны, где такая деятельность осуществляется или создаются предусмотренные договором результаты; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документак договору, заключенному на аукционе, по конкурсу или на бирже, — право страны, где проводится аукцион, конкурс или находится биржа. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаК договорам, не перечисленным в частях пеpвой и втоpой настоящей статьи, при отсутствии соглашения сторон о подлежащем применению праве, применяется право страны, где учреждена, имеет место жительства или основное место деятельности сторона, которая осуществляет исполнение, имеющее решающее значение для содержания такого договора. При невозможности определить исполнение, имеющее решающее значение для содержания договора, применяется право страны, с которой договор наиболее тесно связан.
