Исполнение инкассового поручения Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПри отсутствии какого-либо документа или несоответствии документов по внешним признакам инкассовому поручению исполняющий банк обязан немедленно известить об этом лицо, от которого было получено инкассовое поручение. В случае неустранения указанных недостатков банк вправе возвратить документы без исполнения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаДокументы представляются плательщику в той форме, в которой они получены, за исключением отметок и надписей банков, необходимых для оформления инкассовой операции. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли документы подлежат оплате по предъявлении, исполняющий банк должен сделать представление к платежу немедленно по получении инкассового поручения. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЕсли документы подлежат оплате в иной срок, исполняющий банк должен для получения акцепта плательщика представить документы к акцепту немедленно по получении инкассового поручения, а требование платежа должно быть сделано не позднее дня наступления указанного в документе срока платежа. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаЧастичные платежи могут быть приняты в случаях, когда это установлено банковскими правилами, либо при наличии специального разрешения в инкассовом поручении. Предложения по документуПрослушать аудиоПолучить ссылку из элемента документаПолученные (инкассированные) суммы должны быть немедленно переданы исполняющим банком в распоряжение банку-эмитенту, который обязан зачислить эти суммы на счет клиента. Исполняющий банк вправе удержать из инкассированных сумм причитающиеся ему вознаграждение и возмещение расходов.
