Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадраматические и сценарные произведения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документамузыкальные произведения с текстом и без текста; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документамузыкально-драматические произведения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документахореографические произведения и пантомимы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документааудиовизуальные произведения; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапроизведения живописи, скульптуры, графики, дизайна и другие произведения изобразительного искусства; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапроизведения декоративно-прикладного и сценографического искусства; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапроизведения архитектуры, градостроительства и садово-паркового искусства; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документафотографические произведения и произведения, полученные способами, аналогичными фотографии; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документагеографические, геологические и другие карты, планы, эскизы и произведения, относящиеся к географии, топографии и другим наукам; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документапрограммы для электронно-вычислительных машин (ЭВМ) всех видов, включая прикладные программы и операционные системы; Предложения по документуПрослушать аудиоПолучить ссылку из элемента документадругие произведения, отвечающие требованиям, установленным статьей 1041 настоящего Кодекса.
